```

| № | Название | Длина | Тема | Уровень | Курс (куда отнести) |
|---|----------|-------|------|---------|-------------------|
| 1 | Утренняя виньяса | 10 мин | Энергия | Начин | Утренние комплексы |
| 2 | Здоровая спина | 20 мин | Спина | Сред | Здоровая спина |
| 3 | Уддияна практика | 15 мин | Вес | Сред | Висцеральные практики |
| ... | ... | ... | ... | ... | ... |
```

**Категории для классификации:**

| Курс | Подходящие темы |
|------|-----------------|
| **Здоровая спина** | Спина, шея, осанка, боли в спине |
| **Гормональная йога 45+** | Гормоны, приливы, менопауза, эмоции |
| **Висцеральные практики** | Уддияна, живот, вес, пищеварение |
| **Пранаяма под метроном** | Дыхание, дыхательные практики |
| **Утренние комплексы** | Энергия, бодрость, начало дня |
| **Зен-йога** | Восстановление, релакс, мягкие практики |

### 1.2 Выделить видео для MVP (минимальной версии)

Из 20 видео выбрать **5-7 самых лучших** для запуска сайта:

```
Критерии выбора:
✅ Популярные в TG (много сохранений/репостов)
✅ Разные по времени (5 мин, 10 мин, 20 мин)
✅ Охватывают разные курсы (по 1-2 видео на курс)
✅ Хорошее качество видео и звука
✅ Типичные проблемы аудитории (спина, энергия, гормоны)
```

**Пример MVP-набора:**
```
1. Утренняя практика для энергии — 7 мин (БАЗОВЫЙ уровень)
2. Здоровая спина — 18 мин (СОП уровень)
3. Гормональная йога — 15 мин (СРЕДНИЙ уровень)
4. Пранаяма для стресса — 10 мин (БАЗОВЫЙ)
5. Уддияна практика — 12 мин (СРЕДНИЙ)
```

Остальные 15 видео будут пополнять архив постепенно.
