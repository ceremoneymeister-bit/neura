## СПРАВОЧНАЯ ИНФОРМАЦИЯ

### Уровни:
- **Начин** — для новичков, нет сложных поз
- **Сред** — для среднего уровня, требуется опыт
- **Адв** — продвинутый уровень

### Возможные темы:
- Энергия / утро
- Спина / шея / осанка
- Гормоны / приливы / менопауза
- Вес / уддияна / висцеральные
- Дыхание / пранаяма / метроном
- Восстановление / релакс / нидра
- Стресс / медитация
- Гибкость / растяжка
- Баланс / равновесие
- Иммунитет

### Курсы (куда отнести):
- **Здоровая спина** — всё про спину, шею, осанку
- **Гормональная йога 45+** — гормоны, приливы, эмоции, менопауза
- **Висцеральные практики** — уддияна, живот, вес, пищеварение
- **Пранаяма под метроном** — дыхание, дыхательные техники
- **Утренние комплексы** — энергия, пробуждение, быстрые 5-10 мин
- **Зен-йога** — восстановление, релакс, медитация, нидра

### MVP (первая версия для запуска):
Отметь галочку ☐ → ☑️ для видео, которые должны быть в первые 7 дней на сайте.
Критерии: популярные, разной длины, разные курсы, хорошее качество.
