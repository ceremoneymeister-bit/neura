## 1️⃣ ОБЩАЯ ОЦЕНКА САЙТА (для корпоративного покупателя)

### ❌ КРИТИЧЕСКИЕ ПРОБЛЕМЫ

| Проблема | Влияние | Решение |
|----------|--------|---------|
| **Нет B2B-позиционирования** | Корпоративный покупатель не видит "себя" на сайте | Создать отдельный раздел Corporate Wellness с кейсами компаний |
| **Отсутствуют кейсы / результаты для компаний** | Невозможно оценить ROI для бизнеса | Добавить: "Результаты для организаций" (текучесть, производительность, больничные) |
| **Нет информации о групповых программах** | HR не знает, что можно заказать для коллектива | Описать: корпоративные тарифы, форматы (онлайн/офлайн), гибкость расписания |
| **Нет контактной информации для B2B** | Корпоративный покупатель не знает, как договориться | Добавить форму для корпоративных запросов с отдельным менеджером |
| **Сайт только на русском** | Международные компании не могут разобраться | Минимально: англоязычная версия "Corporate Wellness" раздела |
| **Отсутствует ROI/метрики** | Финансовое обоснование — главное для корпоративных закупок | Показать: снижение больничных дней, улучшение engagement, стоимость vs выгода |

| Проблема | Влияние | Решение |
|----------|--------|---------|
| **Неяснен формат для компаний** | HR не понимает, как это организовать | Расписать: онлайн тарифы, лайв-сессии, записи, расписание для сотрудников |
| **Нет примеров программ** | Сложно представить, что конкретно получится | Добавить: "Типовые программы" (Здоровая спина, Стресс-менеджмент, Энергия женщин 45+) |
| **Фото/видео ориентированы на личный бренд** | Для B2B важны результаты, а не лицо инструктора | Добавить: скрины из групповых занятий, фото участников (анонимные), видео отзывов HR |
| **Нет социальных доказательств от компаний** | Корпоративный покупатель ищет "кто ещё заказывал?" | Добавить: логотипы компаний-партнёров, цитаты HR-менеджеров, кейсы |
| **Слишком персональный тон** | Корпоративный покупатель ищет профессионализма, а не "моя история" | Переписать для B2B: результаты, методика, гарантии, SLA |

## 4️⃣ КОНТЕНТ, КОТОРЫЙ КРИТИЧЕСКИ НЕОБХОДИМ

### 🎯 Раздел "Corporate Wellness" — главный потенциал

**Заголовок (value prop для B2B):**
```
"Йога для сотрудников: меньше стресса, больше энергии, ниже текучесть кадров"
```

**Подблоки:**
1. **"Почему йога в компании работает?"**
   - Факт 1: Сотрудницы 45+ теряют энергию → падает производительность
   - Факт 2: Боли в спине/шее → больничные дни
   - Факт 3: Стресс → текучесть кадров, особенно женщин в период гормональных изменений
   - Решение: Йога для компании = здоровье + стабильность команды + улучшение engagement

2. **"Кейсы компаний"** (ОЧЕНЬ важно для B2B!)
   - Пример 1: "IT-компания, 100 сотрудников"
     - Было: Боли в спине (50% сотрудников), текучесть женщин 45+
     - Стало: Участвуют 30 человек, 80% хотят продолжить, больничные ↓ на 15%
     - Стоимость: 50 000 руб/месяц, экономия на замене кадров: 500 000+

   - Пример 2: "Корпоративный офис, 250 сотрудников"
     - Было: Синдром выгорания, особенно среди женщин-менеджеров
     - Стало: 45 участниц, улучшение mood, снижение стресс-сигналов
     - Сроки: 3 месяца на результаты

   - Пример 3: "Косметическая клиника, 40 сотрудников (все женщины)"
     - Было: Целевая аудитория = точная демография Яны
     - Стало: 25/40 занимаются регулярно, рекомендуют подругам
     - Retention: 98% довольных

3. **"Программы для организаций"** (типовые пакеты)
   - **Базовый (STARTER)**
     - Формат: 2 онлайн-сессии в неделю (утром + вечером)
     - Участников: до 30
     - Стоимость: 30 000 руб/месяц
     - Включает: Живые занятия + запись + Telegram-группа

   - **Стандартный (STANDARD)** — рекомендуемый
     - Формат: 4 онлайн-сессии + 1 офлайн/месяц
     - Участников: до 50
     - Стоимость: 60 000 руб/месяц
     - Включает: Специализированные программы (спина, стресс, женское здоровье)

   - **Премиум (PREMIUM)**
     - Формат: 4 онлайн + 4 офлайн + личные консультации
     - Участников: до 100
     - Стоимость: 120 000 руб/месяц
     - Включает: Кастомизация под проблемы компании + поддержка HR

4. **"Как организуется?"** (практические вопросы)
   - Что нужно от компании? (помещение/зум, список участников)
   - Расписание: можно ли под график компании? (ДА, гибкое)
   - Отчётность: получит ли HR отчёт об участии? (ДА)
   - Программа: какие асаны? все ли могут? (спецпрограммы для спины, стресса)
   - Сроки результатов: когда видны улучшения? (2-4 недели энергия, 2-3 месяца стабильность)

5. **"FAQ для HR"**
   - "Кому подходит? Только женщинам 45+?" → Нет, все возраста, но специализация именно на 45+
   - "Будет ли скучно молодым сотрудникам?" → Программы адаптированы для всех
   - "Как измерить результаты?" → Опросы, боль-шкалы, attendance, feedback
   - "Что если сотрудник не может физически?" → Адаптированные версии для любого уровня
   - "Есть ли гарантии? Страховка?" → Да (страховка для инструктора + гарантия качества)

## 7️⃣ КОПИРАЙТИНГ ДЛЯ B2B (Примеры текстов)

### 📌 Заголовок главной B2B-секции
```
"Йога для женщин 45+ — профилактика, которая экономит компанию деньги"

Ваши сотрудницы 45-60 лет теряют энергию, жалуются на спину и стресс.
Это норма гормональных изменений.

Яна Бережная помогает компаниям включить йогу в систему wellness
и вернуть энергию и стабильность команде.

15+ лет работы с этой возрастной группой.
Результаты видны через 2-4 недели.
```

### 📌 Подраздел "Почему йога в компании экономит деньги"
```
Проблема:
• Болит спина → больничные дни (в среднем -3 дня в год на человека)
• Стресс → текучесть кадров (замена специалиста стоит 3-6 месячных зарплат)
• Энергия упала → производительность ↓ на 20-30%
• Гормональные изменения → нестабильность, конфликты в команде

Решение:
Регулярная йога для женщин 45+ = профилактика здоровья

Результат для компании:
✅ Больничные ↓ на 15-20%
✅ Текучесть женщин 45+ ↓ на 20-30%
✅ Engagement в корпоративных опросах ↑ на 15%
✅ Стоимость программы: 50-100K руб/месяц
✅ Экономия: 200K-500K руб/месяц на замене кадров

Рентабельность: за 2-3 месяца программа окупает себя.
```

   - Переусложнение = потеря контракта

5. **Не забывать про системность**
