# 📚 ГАЙД ПО СБОРУ КЕЙСОВ ДЛЯ ЯНЫ

## 🎯 ШАГ 1: ОПРЕДЕЛИТЬ ПОТЕНЦИАЛЬНЫЕ КЕЙСЫ

### Где искать первые клиенты?

**A. ТЕКУЩИЕ ИЗВЕСТНЫЕ КОНТАКТЫ:**
```
Список людей, которые знают Яну и могут помочь:
- Друзья в компаниях
- Коллеги по йоге (которые работают в офисах)
- Клиентки, которые работают в HR
- Мамины знакомые (если они где-то работают)
- Бывшие коллеги по дизайну/мебели

Действие: Просмотри свои контакты и составь список 5-10 человек,
которых можно попросить либо рекомендацию, либо помочь внедрить программу.
```

**B. ПЕРВЫЕ ПИЛОТ-ПРОГРАММЫ:**
```
Если программ пока нет — СОЗДАТЬ ПИЛОТ:
1. Выбрать одну небольшую компанию (20-30 человек)
2. Предложить пробный месяц со скидкой (50%) или даром
3. Собрать метрики и результаты
4. Переписать в кейс

Это даст:
✅ Реальный кейс с реальными цифрами
✅ Первое социальное доказательство
✅ Опыт, как организовать для корпоративных клиентов
✅ Отзывы от HR-менеджеров
```

**C. ПОТЕНЦИАЛЬНЫЕ НИШИ (легче всего начинать):**
```
🎯 IT-компании
- Проблема: Работают сидя 8+ часов, спина, глаза, стресс
- Решение: Йога для спины + стресс-менеджмент
- HR-фокус: Важна retention (дорогие специалисты)

🎯 Косметические/beauty-компании
- Проблема: Целевая аудитория = точно женщины 40+
- Решение: Гормональная йога = совпадение идеальное
- HR-фокус: Wellness как часть имиджа компании

🎯 Финансовые компании
- Проблема: Стресс, высокая текучесть, гормональные
- Решение: Wellness как retaining tool
- HR-фокус: Бюджет есть

🎯 Медицинские клиники
- Проблема: Врачи и медсестры горят, боли, стресс
- Решение: Йога для восстановления
- HR-фокус: Понимают пользу здоровья

🎯 Франшизы / Салоны красоты
- Проблема: Боли (стоят весь день), текучесть, стресс
- Решение: Йога для спины + восстановления
- HR-фокус: Часто целевая аудитория = женщины
```

## 🎤 ШАГ 2: ПРОВЕСТИ ИНТЕРВЬЮ С HR-МЕНЕДЖЕРОМ

### Как договориться?

**Письмо для контакта:**
```
Тема: Совместный проект — йога для вашей команды

Привет, [Имя]!

Я разработала специальную программу йоги для корпоративных клиентов.
Особенно эффективна для сотрудниц 40+ лет (гормональная йога).

Результаты:
✅ Снижение боль-жалоб (спина, шея)
✅ Увеличение энергии и фокуса
✅ Снижение стресса
✅ 70-80% регулярного участия (сравнено с другими wellness-программами)

Предлагаю БЕСПЛАТНЫЙ пилот на 2-4 недели для вашей команды.
В обмен — реальные метрики и возможность стать кейс-студи (анонимно, если нужно).

Это отличная возможность для компании попробовать без риска,
а для меня — получить первый корпоративный успешный кейс.

Готовы поговорить на 15 минут по Zoom?

С уважением,
Яна Бережная
```

### Интервью: Основные вопросы

**БЛОК 1: ПРОБЛЕМЫ В КОМПАНИИ (5-7 минут)**
```
1. Какие основные wellness-вызовы у вас в компании?
   (Боль, стресс, текучесть, engagement и т.д.)

2. Сколько сотрудниц у вас в возрасте 40+?
   Какие проблемы именно у этой группы?

3. Попробовали ли другие wellness-программы?
   Какие? Почему они не сработали?

4. Какой бюджет у вас на wellness в год?

5. Кто примет решение о новой программе?
   (HR, CEO, CFO)
```

**БЛОК 2: МОТИВАЦИЯ ЗАКАЗАТЬ (5 минут)**
```
1. Почему йога именно сейчас?

2. Что было бы идеальным результатом?

3. Как вы измеряете успех wellness-программы?

4. Есть ли метрики до внедрения программы?
   (Больничные, текучесть, опросы)
```

**БЛОК 3: ПРАКТИКА (5-7 минут)**
```
1. Какой формат предпочесть?
   (Онлайн, офлайн, время дня)

2. Сколько людей хотите вовлечь?
   (20, 50, 100+?)

3. Как долго готовы пилотировать?
   (2 недели, месяц, 3 месяца?)

4. Что нужно от компании для успеха?
   (Напоминания, интеграция в календарь и т.д.)
```

**БЛОК 4: РЕЗУЛЬТАТЫ (если уже была программа)**
```
1. Какие результаты увидели?
   (Метрики, отзывы, изменения)

2. Какой процент участвовал?

3. Какие отзывы получили?
   (Лучше всего подойдут специфичные цитаты)

4. Будете продолжать?
```

## 📊 ШАГ 3: СОБРАТЬ МЕТРИКИ

### Какие данные просить?

**ЕСЛИ БЫЛ ПИЛОТ:**
```
До программы:
- Опрос участников по боли/стрессу (шкала 1-10)
- Данные о больничных дней
- NPS компании (если есть)

Во время программы:
- Attendance rate (% от приглашённых)
- Dropout rate (кто прекратил)

После программы:
- Опрос: "Как изменилась боль? Энергия? Стресс?" (шкала)
- Обратная связь участников (свободный текст)
- Было ли снижение больничных за месяц?
- Желание продолжать (% "да")
```

**ЕСЛИ НОВОГО ПРОГРАММЫ:**
```
Перед внедрением:
- Опрос: "Какие проблемы испытываете?"
  (боль, стресс, энергия, сон) — шкала 1-10
- Текущие больничные (за месяц)
- Текущая текучесть (за 6 месяцев)

После месяца программы:
- Повторный опрос по тем же вопросам
- Attendance
- Qualitative feedback
- "Будете продолжать?" — % ДА

После 3 месяцев:
- Финальные метрики
- Долгосрочные результаты
```

### Как получить отзывы участников?

**Способ 1: Google Form (простой)**
```
Форма для участников:
1. Ваш возраст: ___
2. Как долго занимались с Яной: ___
3. Какая была главная проблема (боль/стресс/энергия)?
4. Как изменилось (1-10 до → после)?
5. Один конкретный результат, который заметили: ____
6. Рекомендовали бы друзьям? (Да/Нет/Может быть)
7. Хотели бы продолжить? (Да/Нет)
```

**Способ 2: Групповой опрос в Telegram**
```
Яна пишет в группу:
"Спасибо за этот месяц! 🙏
Для меня важно понять, что сработало.

Ответьте пожалуйста:
1. На какую оценку (1-10) вы оцениваете программу?
2. Что больше всего помогло? (спина/энергия/сон/стресс)
3. Один совет, что можно улучшить?

Ваши ответы помогут сделать программу ещё лучше!"
```

## 📝 ШАГ 4: ПЕРЕПИСАТЬ В КРАСИВЫЙ КЕЙС

### Структура финального кейса (для сайта / презентации)

```markdown
# КЕЙС: [Компания или "IT-компания с 100 сотрудниками"]

## 📌 Проблема
Сотрудницы 45-55 лет жаловались на:
- Боли в спине (72% участников)
- Усталость и низкая энергия
- Гормональные изменения (приливы, раздражительность)
- Плохой сон

Результат: текучесть женщин 45+ выше на 30%, чем в целом по компании.

## ✅ Решение
Запустили программу йоги с Яной Бережной:
- 2 сессии в неделю онлайн (утро и вечер)
- 35 сотрудниц в пилоте
- 1 месяц программы

## 📊 Результаты
**Через 1 месяц:**
- Attendance: 72% (выше среднего)
- Боли в спине: ↓ с 7.2 → 4.5 (улучшение 37%)
- Энергия: ↑ с 4.1 → 6.8 (улучшение 66%)
- Хотят продолжить: 85%

**Долгосрочный эффект:**
- Через 3 месяца: текучесть женщин 45+ ↓ на 20%
- Больничные дни: ↓ на 15% (сравнено с прошлым периодом)
- Engagement в корпоративных опросах: ↑ на 12%

**Финансово:**
- Инвестиция: 50K руб/месяц
- Сэкономлено на замене 1 кадра (уход не произошёл): ~400K руб
- ROI: программа окупилась за 1.5 месяца

## 💬 Отзывы
**От участниц:**
"Спасибо Яне! Первый раз я почувствовала, что йога — это не про гибкость.
Это про мышцы, про здоровье. Спина реально перестала болеть."
— Елена, 48 лет, специалист

"Я была скептична, но после 2 недель заметила, что проснулась со своей энергией.
Это было невозможно полгода назад."
— Мария, 52 года, менеджер проектов

**От HR:**
"Не ожидали таких результатов. Обычно wellness-программы участвуют 40%.
Здесь 72%. И что важно — результаты видны быстро, не в конце года."
— Ольга, HR-директор

## 🎯 Вывод
Йога для женщин 40+ — это не хобби, это инвестиция в retention и здоровье команды.
Для компании это окупается через 1-2 месяца.
```

## 📌 ПРИМЕЧАНИЯ

**Золотое правило кейсов:**
Реальные цифры + реальные отзывы = доверие покупателя

**Не публиковать:**
- Придуманные результаты
- Обобщённые отзывы ("все очень благодарны")
- Без разрешения компании

**Публиковать (если согласилась):**
- Реальное имя + должность + цифры (если согласна)
- Или анонимно + всё реальное (рекомендуется)
- Специфичные отзывы ("спина с 7 на 4", а не "здоровье улучшилось")
