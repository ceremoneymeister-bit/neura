## ✅ ШАГ 2: ДОБАВИТЬ COOKIE-БАННЕР

### 2.1. Вставить код баннера

**Где:** В файл шаблона сайта (обычно `_footer.html`, `base.html`, `layout.html` или аналог), перед закрывающим тегом `</body>`

**Действие:**
1. Скопировать HTML из файла `cookie-banner-code.html` (начиная с `<div id="cookieBanner"...`)
2. Вставить в `</body>` каждой страницы или в общий шаблон

**Пример:**
```html
</main>
<!-- Вставить здесь -->
<div id="cookieBanner" class="cookie-banner">
    <!-- ... код баннера ... -->
</div>
<!-- Конец вставки -->
</body>
</html>
```

### 2.2. Проверка

При первом посещении сайта (в новом браузере или режиме инкогнито) должен появиться баннер внизу страницы с кнопками:
- `[Принять все]`
- `[Настроить]`
- `[Только необходимые]`

## 🆘 ТИПИЧНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

### Проблема: Яндекс.Метрика загружается ДО согласия

**Решение:**
Убедиться, что в `<head>` нет прямого кода Метрики. Код должен быть в функции `activateYandexMetrika()`.

### Проблема: Чекбокс в форме предустановлен

**Решение:**
Убедиться, что в HTML атрибут `checked` НЕ установлен.
```html
<!-- ❌ НЕПРАВИЛЬНО -->
<input type="checkbox" name="consent" checked>

<!-- ✅ ПРАВИЛЬНО -->
<input type="checkbox" name="consent">
```

### Проблема: Cookie-баннер показывается каждый раз

**Решение:**
Проверить, что браузер может сохранять cookies. Убедиться, что скрипт работает (проверить DevTools → Console на ошибки).

### Проблема: Ссылки из баннера ведут на 404

**Решение:**
Убедиться, что страницы действительно созданы на сайте с корректными URL:
- `/politika-pd/`
- `/soglasie-pd/`
- `/cookie/`
