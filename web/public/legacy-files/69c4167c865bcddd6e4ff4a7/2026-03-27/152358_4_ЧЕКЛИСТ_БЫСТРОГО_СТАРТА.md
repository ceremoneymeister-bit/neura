# ⚡ ЧЕКЛИСТ: СОБРАТЬ ПЕРВЫЕ КЕЙСЫ ЗА 4 НЕДЕЛИ

## 🎯 НЕДЕЛЯ 1: ПОДГОТОВКА

### День 1-2: Определить контакты
- [ ] Просмотреть Телеграм-каналы → кто может знать HR-менеджеров
- [ ] Посмотреть LinkedIn контакты
- [ ] Вспомнить клиентов, которые работают в компаниях
- [ ] Вспомнить знакомых друзей, которые работают в HR/велнесс
- [ ] Составить список 5-10 человек для контакта

### День 3-4: Подготовить предложение
- [ ] Написать письмо с предложением пилот-программы (шаблон выше ↑)
- [ ] Подготовить варианты: 2 недели / 1 месяц / 3 месяца
- [ ] Подготовить варианты цены: 50% скидка на пилот / даром
- [ ] Создать Google Form для опроса участников (шаблон выше ↑)

### День 5-7: Отправить предложения
- [ ] Отправить письма всем 10 контактам (рассылка + личные)
- [ ] Написать в мессенджерах (если есть)
- [ ] Ждать ответов

**РЕЗУЛЬТАТ НЕДЕЛИ 1:** Минимум 2-3 контакта, заинтересованных в пилоте

---

## 🎯 НЕДЕЛЯ 2: ПЕРВЫЕ ИНТЕРВЬЮ

### День 1-2: Договориться на звонок
- [ ] Провести звонки с заинтересованными контактами (15-30 минут)
- [ ] Рассказать о программе
- [ ] Узнать основные проблемы компании
- [ ] Предложить пилот на 2-4 недели
- [ ] Договориться о сроках

### День 3-4: Собрать информацию "ДО"
- [ ] Отправить компании опросник "проблемы участников" (боль, стресс, энергия)
- [ ] Попросить данные о больничных дней (если они готовы)
- [ ] Составить список участников
- [ ] Согласовать расписание занятий

### День 5-7: Запустить первый пилот
- [ ] Провести первое занятие
- [ ] Создать Telegram-группу для участников (если нужно)
- [ ] Отправить ссылку на Google Form опросник
- [ ] Собрать первые отзывы

**РЕЗУЛЬТАТ НЕДЕЛИ 2:** Запущено 1-2 пилот-программы

---

## 🎯 НЕДЕЛЯ 3: СБОР ДАННЫХ (параллельно занятия продолжаются)

### День 1-3: Провести занятия
- [ ] 2-3 занятия за неделю
- [ ] Следить за attendance
- [ ] Собирать обратную связь (устно, в чате)

### День 4-7: Собрать метрики
- [ ] Напомнить участникам про опросник (если не заполняли)
- [ ] Посчитать attendance rate (процент)
- [ ] Выписать отзывы (самые хорошие цитаты)
- [ ] Спросить у HR: "Какие видите результаты?"
- [ ] Собрать данные о больничных (если обещали)

**РЕЗУЛЬТАТ НЕДЕЛИ 3:** Собраны первые данные и отзывы

---

## 🎯 НЕДЕЛЯ 4: ПЕРЕПИСЫВАНИЕ КЕЙСА

### День 1-2: Структурировать данные
- [ ] Открыть "Примеры готовых кейсов" (выше ↑)
- [ ] Выбрать похожий кейс
- [ ] Адаптировать под реальные данные компании

### День 3-4: Переписать кейс
- [ ] Заполнить шаблон кейса реальными цифрами
- [ ] Переписать в "красивый" формат (как примеры выше ↑)
- [ ] Добавить реальные отзывы
- [ ] Рассчитать финансовый эффект (ROI)

### День 5-6: Согласовать с компанией
- [ ] Отправить черновик кейса HR-менеджеру
- [ ] Спросить: "Можно ли это опубликовать?"
- [ ] Если нет → анонимизировать (убрать название компании)
- [ ] Получить подтверждение

### День 7: Финализировать
- [ ] Сделать финальную версию кейса
- [ ] Добавить фото (если компания разрешила)
- [ ] Сохранить в папку "кейсы"

**РЕЗУЛЬТАТ НЕДЕЛИ 4:** Готовый кейс (1-3 примера)

---

## 📊 ПАРАЛЛЕЛЬНО: СОБИРАТЬ КОНТАКТЫ НА БУДУЩЕЕ

### Постоянные действия (каждый день):
- [ ] Добавлять в список контакты HR-менеджеров
- [ ] Отслеживать потенциальных клиентов в соцсетях
- [ ] Работать с теплыми контактами (те, кто уже знает про Яну)

### Рекомендуемый источник контактов:
1. **LinkedIn** — найти HR-директоров (по городу, по компаниям)
2. **Друзья и клиенты** — они лучше всего рекомендуют
3. **Профессиональные сообщества** — HR-бургер, wellness-клубы
4. **Холодные звонки** — в HR отделы компаний (40-50 человек)

## 💰 ФИНАНСОВЫЙ РАСЧЁТ: СКОЛЬКО МОЖЕТ ЗАРАБОТАТЬ ЯНА

### Сценарий 1: Консервативный (реалист)
```
Собрать 3 корпоративных клиента в первый год:
- Контракт 1: 50K руб/месяц х 6 месяцев = 300K
- Контракт 2: 60K руб/месяц х 8 месяцев = 480K
- Контракт 3: 100K руб/месяц х 4 месяца = 400K

ИТОГО за год: 1,180,000 руб

Плюс: личные клиенты (как сейчас) = 300-500K
ВСЕГО за год: 1,480,000 - 1,680,000 руб
```

### Сценарий 2: Оптимистичный
```
5-7 корпоративных контрактов + личные клиенты:
ВСЕГО за год: 2,000,000+ руб

Плюс: масштабирование (добавить инструкторов):
ВСЕГО за год: 5,000,000+ руб
```

### Сценарий 3: Долгосрочный (2-3 года)
```
Система из 3-4 инструкторов под руководством Яны
Корпоративные программы для 10+ компаний
Годовой доход: 3-10M руб
```

## 🚀 КАК УСКОРИТЬ РЕЗУЛЬТАТ

### Тактика 1: Холодные звонки в HR
```
Найти 50 компаний (IT, медицина, дизайн, финансы)
Позвонить в HR
Предложение: "Пилот 2 недели со скидкой 50%"
Conversion: 10-20% = 5-10 контактов
```

### Тактика 2: Рефереральная программа
```
"Порекомендуй HR-менеджера, получи скидку 30% на свои занятия"
Это сработает, если у Яны уже есть 100+ личных клиентов
```

### Тактика 3: Льготный пилот
```
Предложить первым 5 компаниям:
- 1 месяц программы = 50% цены (вместо полной)
- Обещание: будут в примерах/кейсах
- Результат: 5 первых клиентов за 2-3 месяца
```

## ⚠️ ЧАСТЫЕ ОШИБКИ (ИЗБЕЖАТЬ)

❌ **ОШИБКА 1:** Собирать только хорошие отзывы
✅ Собирать реальные отзывы (и хорошие, и "нейтральные")

❌ **ОШИБКА 2:** Скрывать трудные моменты
✅ Если dropout 20% — это нормально и честно (лучше чем скрывать)

❌ **ОШИБКА 3:** Выдумывать цифры
✅ Реальные цифры работают в 100 раз лучше

❌ **ОШИБКА 4:** Долгие интервью
✅ 15-30 минут максимум (HR занят)

❌ **ОШИБКА 5:** Не получать разрешение на публикацию
✅ Всегда спросить: "Могу ли я это показать другим компаниям?"

❌ **ОШИБКА 6:** Собирать данные только "после"
✅ Собирать данные "ДО" и "ПОСЛЕ" (это даёт контраст)

## 💡 ГЛАВНАЯ ИДЕЯ

**Один реальный кейс = 10 холодных звонков**

Если у тебя есть реальный пример:
- IT-компания (100 чел) с результатами
- Косметическая клиника (40 чел) с результатами

То продавать намного проще.

Поэтому первый кейс — самый важный.
Вкладываем 4 недели → получаем его → потом это окупается 100 раз.

## 🎁 БОНУС: КОТОРАЯ КОМПАНИЯ ПЕРВАЯ

**Лучше всего подходят:**
1. Компания, которую ты уже знаешь (есть контакт)
2. Небольшая компания (20-50 человек) — проще согласиться
3. Компания с wellness-бюджетом (IT, финансы)
4. Компания с женским коллективом (красота, медицина)

**Худший вариант:**
- Холодная рассылка большой компании (низкий conversion)
- Компания, которая только говорит "может быть"
- Компания без бюджета на wellness

Выбери первого клиента из контактов, которые тебя уже знают! 🎯
