# Чеклист кастомизации под клиента

## Перед началом

- [ ] Получить от клиента: название, цвет, логотип, структуру курса, тарифы
- [ ] Определить домен для деплоя
- [ ] Клонировать шаблон: `cp -r projects/course-platform/ projects/Producing/{Client}/platform/`

## Обязательные изменения

### 1. config.js
- [ ] `name` — название платформы
- [ ] `subtitle` — подзаголовок
- [ ] `author` — имя автора/эксперта
- [ ] `accent` — акцентный HEX-цвет
- [ ] `telegram` — ссылка на канал/чат
- [ ] `email` — контактный email
- [ ] `domain` — домен сайта

### 2. index.css @theme
- [ ] `--color-accent` — основной акцентный цвет
- [ ] `--color-accent-light` — светлый вариант акцента (hover)

### 3. data/courses.js
- [ ] Заменить курс на реальный
- [ ] Название, описание курса
- [ ] Модули: названия, описания
- [ ] Уроки: названия, длительность, контент
- [ ] videoUrl: ссылки на видео (или null для placeholder)

### 4. data/pricing.js
- [ ] Количество тарифов (1-3)
- [ ] Названия, цены, описания
- [ ] Списки фич для каждого тарифа
- [ ] Какой highlighted (рекомендуемый)

### 5. data/testimonials.js
- [ ] Заменить на реальные отзывы ИЛИ удалить секцию из LandingPage

### 6. Брендинг
- [ ] `public/favicon.svg` — логотип клиента
- [ ] `index.html` → `<title>` — название платформы

## Опциональные изменения

### Landing секции
- [ ] Hero.jsx — заголовок, подзаголовок, CTA
- [ ] Problems.jsx — боли целевой аудитории
- [ ] ForWhom.jsx — описание ЦА
- [ ] FAQ.jsx — реальные вопросы по продукту
- [ ] CTA.jsx — финальный призыв

### Удаление ненужного
- [ ] Testimonials — если нет отзывов
- [ ] Problems — если не нужен блок проблем
- [ ] ForWhom — если ЦА очевидна
- [ ] AdminPage — если не нужна админка

## Верификация перед деплоем

- [ ] `npm run dev` — все страницы открываются
- [ ] Брендинг корректный (цвет, лого, название)
- [ ] Лендинг: все секции актуальны
- [ ] Авторизация: login → cabinet
- [ ] Курсы: навигация course → modules → lessons → video
- [ ] Прогресс: отметить урок → рефреш → сохранён
- [ ] Мобильная версия: hamburger, адаптивная сетка
- [ ] `npm run build` без ошибок
- [ ] `.htaccess` в `dist/`

## Деплой

```bash
npm run build
# Скилл regru — читай .agent/skills/regru/SKILL.md
python3 scripts/ftp-deploy.py --source dist/ --target /public_html/
```

## Типичные ошибки клиентов

| Ошибка | Симптом | Решение |
|--------|---------|---------|
| Забыли сменить accent в CSS | Кнопки indigo, а лого красное | Обновить --color-accent в index.css |
| Оставили mock testimonials | "Александр М., Предприниматель" | Заменить или удалить секцию |
| videoUrl не заменили | Все уроки показывают placeholder | Залить видео, обновить ссылки |
| Не обновили title | "Обучающая платформа" в табе | index.html → `<title>` |
