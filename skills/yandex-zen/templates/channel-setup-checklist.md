# Чеклист: Запуск канала на Яндекс Дзен

> Пошаговый план от нуля до первых 10 статей и заявки на монетизацию. Каждый шаг — конкретное действие с критериями выполнения.

---

## Когда использовать

- Запускаете новый канал на Дзене с нуля
- Перезапускаете заброшенный канал
- Настраиваете канал для клиента
- Нужен системный подход вместо хаотичных публикаций

---

## Фаза 1: Подготовка (до создания канала)

### 1.1. Выбор ниши

- [ ] Определить тему канала (одна ниша, не «обо всём»)
- [ ] Проверить спрос: найти 3-5 каналов в нише через поиск Дзена
- [ ] Оценить конкуренцию: сколько подписчиков у топовых каналов
- [ ] Найти 3-5 каналов-конкурентов в Telegram через TGStat — отсортировать посты по репостам и реакциям
- [ ] Выписать 10 тем, которые сработали лучше всего у конкурентов
- [ ] Сформулировать УТП канала: «Чем мой канал отличается от [КОНКУРЕНТ]?»

**Критерий выполнения:** записана ниша, 10 работающих тем, сформулировано УТП в 1-2 предложениях.

### 1.2. Портрет аудитории

- [ ] Описать целевого читателя: возраст, пол, интересы, боли
- [ ] Учесть ядро Дзена: женщины 50+ (но не ограничиваться)
- [ ] Определить формулу коннекта: эмоции + узнавание себя + надежда
- [ ] Выбрать тон общения: «Как я разговариваю с этим человеком лично?»
- [ ] Записать 5 главных болей аудитории

**Критерий выполнения:** заполнен портрет ЦА, 5 болей, выбран тон.

### 1.3. Контент-стратегия

- [ ] Выбрать основной формат: списки / how-to / рассказы / обзоры
- [ ] Запланировать частоту публикаций: минимум 3 статьи в неделю на старте
- [ ] Подготовить контент-план на первые 2 недели (10 тем с заголовками)
- [ ] Для каждой темы определить формат и примерный объём
- [ ] Создать рубрику-«сериал» (статьи, из которых читатель переходит по ссылке дальше)

**Критерий выполнения:** готов контент-план на 10 статей с заголовками и форматами.

---

## Фаза 2: Создание и настройка канала

### 2.1. Регистрация

- [ ] Зайти на dzen.ru, авторизоваться через Яндекс ID
- [ ] Перейти в «Дзен Студию» (studio.dzen.ru)
- [ ] Нажать «Создать канал»
- [ ] Выбрать название канала (запоминающееся, отражает нишу, до 30 символов)

### 2.2. Оформление канала

- [ ] Загрузить аватарку (читается в маленьком кружке, контрастная)
- [ ] Написать описание канала: кто автор, о чём канал, почему стоит подписаться (до 200 символов)
- [ ] Указать ссылки на соцсети (если есть)
- [ ] Добавить обложку канала (1200x400 px, контрастная)

### 2.3. Технические настройки

- [ ] Включить комментарии (для вовлечённости)
- [ ] Настроить фильтр комментариев (от спама)
- [ ] Подключить Яндекс Метрику (для аналитики)
- [ ] Проверить, что канал виден в поиске

**Критерий выполнения:** канал создан, оформлен, настройки выставлены.

---

## Фаза 3: Первые публикации (статьи 1-5)

### 3.1. Подготовка к первой статье

- [ ] Выбрать самую сильную тему из контент-плана
- [ ] Написать 2-3 варианта заголовка (до 60 символов каждый)
- [ ] Подготовить обложку (яркая, контрастная, читается с телефона, уникальная)
- [ ] Подготовить 3-4 уникальные картинки для статьи

### 3.2. Написание статьи

- [ ] Объём: 3000-5000 символов
- [ ] Структура: вступление с хуком -> основной контент -> заключение с вопросом
- [ ] Проверить уникальность текста: от 85% (text.ru или аналог)
- [ ] Прогнать через AI-детектор: ниже 30% (kanalkapital.ru/detector или аналог)
- [ ] Убрать AI-штампы: «важно отметить», «в современном мире», «стоит подчеркнуть» и т.д.
- [ ] Добавить личный опыт или мнение (от первого лица)
- [ ] Добавить минимум 1 диалог или цитату

### 3.3. Публикация и замеры

- [ ] Опубликовать первую статью
- [ ] Записать время публикации
- [ ] Через 24 часа: зафиксировать показы, дочитывания, подписки
- [ ] Через 48 часов: оценить динамику
- [ ] Повторить для статей 2-5 (одна статья через день минимум)

**Критерий выполнения:** 5 статей опубликованы, статистика записана для каждой.

---

## Фаза 4: Наращивание (статьи 6-10)

### 4.1. Анализ первых результатов

- [ ] Какая из 5 статей набрала больше всего дочитываний?
- [ ] Какой формат/тема сработал лучше?
- [ ] Откуда трафик: рекомендации, поиск, подписчики?
- [ ] Средний процент дочитываемости (цель: от 40%)
- [ ] Есть ли комментарии? Какие?

### 4.2. Корректировка стратегии

- [ ] Удвоить ставку на тему/формат, который сработал
- [ ] Отказаться от того, что не зашло (или изменить подход)
- [ ] Добавить внутренние ссылки: из новых статей — на первые пять
- [ ] Создать подборку из первых статей (рубрика)
- [ ] Протестировать другое время публикации

### 4.3. Публикация статей 6-10

- [ ] Каждая статья: по шаблону из фазы 3.2
- [ ] Минимум 1 статья — «сериал» (продолжение предыдущей, с ссылкой)
- [ ] Минимум 1 статья — другой формат (если первые 5 были списки — попробовать рассказ)
- [ ] Отвечать на все комментарии (алгоритм учитывает активность автора)
- [ ] Каждую статью записать в таблицу аналитики

**Критерий выполнения:** 10 статей опубликованы, определён работающий формат, есть внутренние ссылки.

---

## Фаза 5: Монетизация

### 5.1. Условия подключения монетизации (актуально на 2026)

- [ ] Проверить текущие требования в Дзен Студии (условия обновляются)
- [ ] Ориентир: не менее 10 000 минут просмотра за последние 7 дней ИЛИ 10 000 дочитываний статей
- [ ] Канал должен соответствовать правилам площадки (нет нарушений)
- [ ] Контент уникальный, без признаков спама

### 5.2. Подача заявки

- [ ] Перейти в раздел «Монетизация» в Дзен Студии
- [ ] Подать заявку на подключение программы для авторов
- [ ] Привязать кошелёк (банковская карта / ЮMoney)
- [ ] Дождаться модерации (от 1 до 7 дней)
- [ ] После одобрения: проверить, что реклама отображается в статьях

### 5.3. Оптимизация дохода

- [ ] Продолжать публиковать минимум 3 статьи в неделю
- [ ] Фокус на дочитываемость (от этого зависит доход)
- [ ] Тестировать длинные статьи (4000-5000 символов — больше рекламных блоков)
- [ ] Отслеживать RPM (доход на 1000 показов) еженедельно

**Критерий выполнения:** монетизация подключена, первый доход зафиксирован.

## Антипаттерны (чего НЕ делать при запуске)

1. **Публиковать всё сразу.** 10 статей за день — алгоритм воспримет как спам. Разносите публикации: минимум 1 день между статьями, лучше — через день

2. **Менять нишу после 3 статей.** «Не зашло — попробую другое» — дайте алгоритму время. Минимум 10 статей в одной теме, прежде чем делать выводы

3. **Игнорировать аналитику.** Публиковать и не смотреть на цифры — пустая трата времени. Записывайте показы, дочитывания, подписки для каждой статьи

4. **Копировать чужие статьи.** Уникальность ниже 85% = бан в рекомендациях. Это правило Дзена, и оно жёсткое

5. **Без обложки.** Статья без яркой обложки = CTR в 2-3 раза ниже. Каждая статья — с уникальной контрастной картинкой

6. **Не отвечать на комментарии.** Алгоритм учитывает активность автора. Нет ответов = меньше показов. Отвечайте на каждый комментарий

7. **Ждать подписчиков перед монетизацией.** Работайте на дочитывания — подписки придут. Формула: качественный контент -> дочитывания -> рекомендации -> подписки

8. **Использовать AI-тексты без редактуры.** Дзен с 2025 года отслеживает AI-контент (150+ маркеров). Всегда правьте, добавляйте личный опыт, прогоняйте через детектор
