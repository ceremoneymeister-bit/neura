# Изображения в Дзене: Обложки, картинки в статьях, требования

## Обзор

Визуальная составляющая на Дзене напрямую влияет на CTR карточки в ленте и дочитываемость статьи. Обложка — первое, что видит пользователь вместе с заголовком, и именно она определяет, кликнет он или пролистает дальше.

---

## 1. Обложка статьи: технические требования

### Размеры и пропорции

| Параметр | Значение |
|----------|---------|
| Рекомендуемое разрешение | 1920 x 1080 px |
| Минимальное разрешение | 960 x 540 px |
| Соотношение сторон (лента) | 16:9 |
| Соотношение сторон (подписки/подборки) | 4:3 |
| Безопасная зона (центр) | 1440 x 1080 px (4:3) |
| Максимальный вес файла | до 10 МБ |
| Форматы | JPG, PNG, BMP, SVG |

**Важно:** Дзен использует два вида карточек в зависимости от контекста показа:
- **16:9** — основная лента, главная страница канала, мобильные устройства
- **4:3** — лента подписок, подборки

Ключевая часть изображения должна вписываться в прямоугольник 4:3 (960x720 px), расположенный по центру. Всё, что за пределами этой зоны, может обрезаться в некоторых местах показа.

### Как назначается обложка
В Дзене нельзя загрузить произвольную обложку с компьютера. Обложкой становится одна из картинок, вставленных в текст статьи. По умолчанию — первое изображение. Если картинок в статье нет, вместо обложки отобразится серый квадрат, что критически снижает CTR.

### A/B-тестирование обложек
Дзен поддерживает нативное тестирование: можно выбрать до **3 обложек** и **5 заголовков**. Алгоритм будет показывать разные комбинации и автоматически оставит самую кликабельную карточку. Это бесплатный и мощный инструмент — используйте его для каждой публикации.

## 2. Картинки внутри статьи: требования

| Параметр | Значение |
|----------|---------|
| Минимальная ширина | 400 px |
| Максимальный вес | 20-30 МБ (разные источники) |
| Форматы | JPEG, PNG, GIF, WebP |
| Ориентация | Горизонтальная или квадратная (предпочтительно) |

### Правила для картинок в тексте

- **Горизонтальные или квадратные** — предпочтительны. Вертикальные картинки обрезаются некрасиво и занимают много экранного пространства, снижая дочитываемость.
- Картинки можно сопровождать **подписями** (caption) — используйте эту возможность для пояснений и микро-CTA.
- Не перегружайте статью картинками ради картинок — каждое изображение должно нести смысловую нагрузку.
- Для монетизации: статьи с 5+ вертикальными картинками + 3500 символов текста получают второй рекламный блок.

## 3. Какие обложки дают больше кликов

### Принцип «Отражение реальности»
Читатель смотрит на обложку с вопросом: «Это похоже на то, что я вижу вокруг себя?». Обложка должна резонировать с его повседневным опытом. Абстрактные стоковые фото работают хуже, чем реалистичные ситуативные кадры.

### Что работает:

1. **Люди и эмоции.** Лица с выраженными эмоциями (удивление, радость, задумчивость) привлекают внимание. Но — не стоковые улыбки, а естественные выражения.

2. **Контраст и яркость.** В ленте Дзена много контента — чтобы выделиться, нужен визуальный контраст. Яркие цвета, контрастные сочетания.

3. **Минимум деталей.** Обложка не должна быть перегружена мелкими элементами — поверх изображения Дзен накладывает заголовок, и мелкие детали потеряются.

4. **До/после.** Коллажи с наглядным сравнением результата — один из самых кликабельных форматов.

5. **Цифры и текст на обложке.** Крупные числа или 2-3 слова текста на самом изображении могут усилить кликабельность, но текст должен быть читаемым на мобильных устройствах.

6. **Интрига без кликбейта.** Обложка должна создавать вопрос, но не обманывать. Алгоритм учитывает корреляцию CTR и дочитываемости — высокий CTR + низкая дочитываемость = штраф.

### Что НЕ работает:

- Стоковые фотографии «из коробки» (бизнесмен жмёт руку, девушка с ноутбуком)
- Мелкий неразборчивый текст на картинке
- Тёмные, мутные изображения
- Скриншоты без обработки (мелкий текст, интерфейсные элементы)
- Вертикальные фото (обрезаются в карточке)
- Логотипы без контекста

## 4. Инструменты для создания обложек

### Бесплатные:
- **Canva** — шаблоны 1920x1080, готовые макеты для блогов
- **Figma** — для продвинутых пользователей, точный контроль размеров
- **Photopea** — онлайн-аналог Photoshop
- **Remove.bg** — удаление фона для коллажей

### Генерация через AI:
- **Midjourney / DALL-E / Kandinsky** — генерация уникальных иллюстраций
- Важно: AI-картинки допустимы, но должны выглядеть уместно. Явно «синтетические» обложки могут снижать доверие.

### Рекомендуемый workflow:
1. Создать шаблон 1920x1080 с безопасной зоной 4:3 по центру
2. Разместить ключевой визуал в безопасной зоне
3. Добавить 2-3 слова текста (опционально)
4. Экспортировать в JPEG с качеством 80-90% (оптимальный баланс вес/качество)
5. Загрузить первой картинкой в статью
6. Воспользоваться A/B-тестированием (3 варианта обложек)

## 5. Ограничения платформы по изображениям

### Запрещённый контент на обложках (правила Дзена):
- Шок-контент, насилие, кровь
- Обнажённая натура и эротика
- Элементы, вводящие в заблуждение (фальшивые кнопки play, стрелки, крестики)
- Обещания, не соответствующие содержанию статьи
- Изображения несовершеннолетних в коммерческом контексте

### Технические ограничения:
- Обложка выбирается только из картинок в статье — нельзя загрузить отдельно
- GIF-файлы отображаются как статичная картинка на обложке
- Анимация в обложке не поддерживается
- При кросспостинге из Telegram медиафайлы переносятся, если их размер не превышает 20 МБ

## 6. Чек-лист для обложки

- [ ] Разрешение минимум 960x540, идеально 1920x1080
- [ ] Ключевой элемент — в центральной зоне 4:3
- [ ] Горизонтальная ориентация
- [ ] Яркость и контраст достаточны для выделения в ленте
- [ ] Нет мелких деталей, которые потеряются при уменьшении
- [ ] Текст на картинке (если есть) — крупный, читается на мобильном
- [ ] Соответствует содержанию статьи (не кликбейт)
- [ ] Формат JPEG или PNG, вес до 10 МБ
- [ ] Загружена в статью первой картинкой
- [ ] Подготовлены 2-3 варианта для A/B-теста
