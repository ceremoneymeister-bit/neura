# Требования к помещению {{SCHOOL_NAME}}

## Обязательные критерии

| Требование | Минимум | Идеал |
|------------|---------|-------|
| Площадь | 20 м² | 30-40 м² |
| Вместимость | 7 человек (6 детей + учитель) | 10+ (запас для роста) |
| Wi-Fi | 100 Мбит/с | 300+ Мбит/с |
| Розетки | 8 шт | 12+ шт |
| Стулья | Удобные, с регулировкой высоты | Эргономичные |
| Столы | Отдельные или общий большой | Отдельные рабочие места |
| Освещение | Достаточное для работы за экраном | Регулируемое |
| Кондиционер/отопление | Есть | Есть + тихий |
| Туалет | В здании | На этаже |
| Доступность | Общественный транспорт рядом | Метро + парковка |

## Желательные критерии

- [ ] Проектор или ТВ для демонстрации
- [ ] Кухня / зона с водой (для перерывов)
- [ ] Тихая зона (не шумные соседи)
- [ ] Красивый интерьер (для фото/видео контента)
- [ ] Возможность брендирования (наклейки, баннер)
- [ ] Безопасный вход (не надо проходить через бар/ночной клуб)

|-----|----------|---------------------|

| Учебный зал | 350-600₽ | 5 600 - 9 600₽ |
