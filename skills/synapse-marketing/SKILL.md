## Ключевые дифференциаторы (использовать в КАЖДОМ материале)

1. **AI-мышление, не кодинг.** Учим ставить задачи AI, критически оценивать результаты, создавать системы — не писать код
2. **Микрогруппы 4-6 человек.** У конкурентов 10-15 — у нас персональное внимание
3. **Подписки на 10 000 руб. включены.** ChatGPT Pro, Midjourney, Claude Pro, Cursor — родителям не нужно ничего докупать
4. **Портфолио на выходе.** Каждый ученик создаёт реальные проекты, не решает упражнения
5. **Практикующие AI-специалисты.** Не учителя информатики — люди, которые работают с AI ежедневно
6. **0 прямых конкурентов** в Новосибирске в формате офлайн AI-мышления

- **Для родителей:** спокойный, уверенный, экспертный. Факты и цифры. Без давления — через ценность

- **Бренд-стиль:** neo-brutalist, acid lime на чёрном, Space Grotesk + Inter

## Продуктовая матрица (воронка НЕЧ20)

Каждый уровень продуктовой лестницы решает свою задачу в воронке. Использовать при планировании офферов и upsell-цепочек.

| Уровень | Продукт | Цена | Задача |
|---------|---------|------|--------|
| **Лид-магнит** | Бесплатный мастеркласс «Первый AI-проект за 1 час» | 0₽ | Привлечь контакт, показать ценность |
| **Трипвайр** | Пробное занятие (1 урок с готовым результатом) | 500–990₽ | Снять барьер первого платежа, дать опыт |
| **Основной продукт** | Модуль/курс (8–12 занятий) | 15 000–25 000₽ | Основной доход, глубокое обучение |
| **Максимизатор** | Индивидуальные занятия, VIP-группы, летний AI-лагерь | 40 000₽+ | Увеличить средний чек с готового клиента |
| **Удержание** | Ежемесячный AI-клуб, хакатоны, сертификация | 5 000–10 000₽/мес | LTV, сообщество, рекуррентный доход |

**Принцип:** каждый продукт — самостоятельная ценность. Ученик получает результат на каждом уровне, а не «тизер» следующего.

## 6 шаблонов воронок (НЕЧ20)

Выбирать воронку в зависимости от канала привлечения и стадии проекта.

### 1. Qualifying (квалификация через квиз)
```
Реклама → Квиз «Подходит ли AI-школа вашему ребёнку?» → Персональная рекомендация → Запись на пробное
```
**Когда:** холодный трафик, нужна сегментация. Лучший вариант для Яндекс.Директ.

### 2. Paid Entry (платный вход)
```
Реклама → Пробное занятие 990₽ → Основной курс
```
**Когда:** тёплая аудитория, нужны только мотивированные родители. Отсекает «посмотрю и забуду».

### 3. Recurring (подписка)
```
Пробное → Абонемент (автопродление) → Доп. модули
```
**Когда:** стабильный поток учеников, задача — удержание и предсказуемый доход.

### 4. Full (полная воронка)
```
Бесплатный МК → Пробное занятие → Основной курс → AI-клуб
```
**Когда:** запуск с нуля, максимальный охват. Длинная, но самая конверсионная.

### 5. Webinar (вебинар для родителей)
```
Регистрация → Вебинар «AI и будущее вашего ребёнка» → Запись на пробное
```
**Когда:** масштабирование, онлайн-аудитория, можно привлекать из других городов.

### 6. Cascade (каскадный вход)
```
Мини-курс (3 занятия) → Полный курс → Летний AI-лагерь
```
**Когда:** каникулярные периоды, родители хотят «попробовать немного» перед обязательствами.

## Квиз-архитектура для квалификации родителей (НЕЧ20)

5 типов вопросов квиза → персонализированная рекомендация программы. Использовать в Qualifying-воронке.

### Структура вопросов

| # | Тип вопроса | Пример | Что даёт |
|---|-------------|--------|----------|
| 1 | **Возраст ребёнка** | «Сколько лет вашему ребёнку?» (9-11 / 12-14) | Маршрутизация по возрастным группам |
| 2 | **Опыт с технологиями** | «Ваш ребёнок уже пользуется ChatGPT / Midjourney?» (да/нет/не знаю) | Выбор уровня сложности программы |
| 3 | **Цель родителей** | «Что важнее всего?» (развитие мышления / будущая профессия / полезное хобби) | Акценты в коммуникации после квиза |
| 4 | **Бюджет и формат** | «Какой формат удобнее?» (офлайн группа / онлайн / индивидуально) | Подбор продукта из матрицы |
| 5 | **Срочность** | «Когда хотите начать?» (на этой неделе / через месяц / пока изучаю) | Приоритет в обработке лидов |

### Страница результата

После прохождения — персонализированная рекомендация:
- Название подходящей программы + описание
- Ближайшая дата старта группы
- Бонус за прохождение квиза (скидка 10% на пробное / PDF-гайд / приоритетная запись)
- CTA: «Записать ребёнка» + номер для связи с Владом

**Антипаттерн:** квиз НЕ должен быть длиннее 5 вопросов — каждый дополнительный вопрос минус 15-20% к доходимости.

## Декомпозиция цели набора (обратная воронка, НЕЧ20)

Формула: **Доход = Цена × Количество**. Считаем сверху вниз.

### Пример: цель — 400 000₽/мес

```
🎯 Цель:          400 000₽/мес
├── Учеников:     20 × 20 000₽
├── Пробных:      40 (конверсия пробное → курс: 50%)
├── Заявок:       200 (конверсия заявка → пробное: 20%)
└── Охват:        2 000 (конверсия охват → заявка: 10%)
```

### Как использовать

1. **Подставить реальные цены** из SYNAPSE_CANONICAL.md
2. **Замерить конверсии** после первых 2-3 потоков и скорректировать
3. **Разбить охват по каналам:** Яндекс.Директ (1 000) + VK Ads (500) + Instagram (300) + сарафан (200)
4. **Рассчитать бюджет:** стоимость лида × количество лидов = рекламный бюджет
5. **Отслеживать еженедельно:** если конверсия падает — искать узкое место

**Правило:** если конверсия пробное → курс ниже 30%, проблема в качестве пробного занятия, а не в рекламе.

## 17-шаговый фреймворк запуска набора (Пыриков)

Адаптация под запуск набора в новый поток. Использовать за 4-6 недель до старта занятий.

### Фазы запуска

**Фаза 0 — Подготовка (за 6 недель)**
1. ✅ Определить даты: старт набора → дедлайн записи → первое занятие
2. ✅ Зафиксировать оффер: программа + цена + бонусы ранней записи
3. ✅ Собрать базу: все контакты с прошлых МК, заявки, подписчики

**Фаза 1 — Прогрев (2 недели до открытия записи)**
4. 📢 Контент «проблема»: серия постов о том, зачем детям AI-мышление
5. 📢 Контент «социальное доказательство»: отзывы, результаты учеников, кейсы
6. 📢 Контент «закулисье»: подготовка к новому потоку, обновления программы
7. 📢 Анонс-тизер: «скоро открываем набор — следите»

**Фаза 2 — Открытие записи (1 неделя)**
8. 🚀 Анонс открытия: пост + рассылка + реклама. Бонус ранней записи (скидка / подарок)
9. 🚀 Мастеркласс: бесплатный МК как главный конверсионный инструмент
10. 🚀 Обработка заявок: Влад звонит в течение 24 часов по скрипту `enrollment-call.md`

**Фаза 3 — Основная продажа (1 неделя)**
11. 💰 Кейсы и результаты: «вот что создали ученики прошлого потока»
12. 💰 Работа с возражениями: контент по `parent-objections.md`
13. 💰 Дефицит: «осталось X мест» (реальное, микрогруппы 4-6 = честный дефицит)
14. 💰 Дедлайн: последний день записи с бонусом

**Фаза 4 — Закрытие и старт (финал)**
15. 🏁 Последний призыв: «сегодня последний день записи по спеццене»
16. 🏁 Приветствие записавшихся: инструкции, чат группы, расписание
17. 🏁 Ретроспектива запуска: что сработало, что нет → обновить скилл

**Ключевой принцип:** 80% продаж происходят в последние 48 часов. Не паниковать, если в начале записи тихо.
