# Platform Adaptation — адаптация голоса под платформы

## Принцип

Голос (лексика, тон, персона, эмоции) остаётся неизменным.
Адаптируется только формат: длина, структура, форматирование, хештеги.

---

## Telegram

**Формат — полный, родной:**
- Markdown: **bold**, _italic_, `code`, [ссылки](url)
- Длина: без ограничений, но оптимально 500-2000 символов
- Абзацы: разделять пустой строкой, не стеной текста
- Хештеги: НЕ используются (это не Instagram)
- Ссылки: inline в тексте, не отдельным блоком
- Эмодзи: по профилю автора (кто-то ставит много, кто-то ноль)
- Медиа: изображение/видео усиливает пост, но не обязательно

**Что сохраняется из профиля:**
- Всё: хуки, структура, тон, CTA, лексика, фирменные приёмы

---

## VK (ВКонтакте)

**Адаптация от Telegram:**
- Чуть формальнее: VK-аудитория шире, меньше нишевого сленга
- Хештеги допустимы: 3-5 штук, в конце поста
- Длинные посты ОК, но первые 2 строки решают (кат "Показать полностью")
- Ссылки: можно в тексте, VK не режет охват за внешние ссылки
- Форматирование: **bold** через спец-символы или без, VK ограничен
- Опросы и голосования: нативная фича VK, использовать для engagement

**Что адаптируется:**
- Markdown → plain text с эмодзи-маркерами (→, •, ▪)
- Специфический TG-сленг → более универсальные выражения
- Inline ссылки → полные URL или кнопки

**Что сохраняется:**
- Тон, энергия, персона
- Структура постов (хук → тело → CTA)
- Лексические маркеры
- Фирменные приёмы

---

## Instagram

**Адаптация от Telegram:**
- Лимит: 2200 символов caption (оптимально 500-1000)
- Хештеги: блок из 15-30 в конце или первом комментарии
- Больше эмодзи: Instagram-аудитория ожидает визуальность даже в тексте
- Первая строка: критична — видна до "ещё", максимум 125 символов
- Форматирование: нет bold/italic, структура через переносы строк и эмодзи
- Ссылки: НЕ кликабельны в caption, направлять в bio/stories

**Что адаптируется:**
- Длина: сокращать, убирать детали → оставлять суть
- Bold/italic → CAPS для акцентов (умеренно) или эмодзи-маркеры
- Inline ссылки → "Ссылка в bio" / "Подробнее в stories"
- Markdown → plain text с эмодзи-структурой
- Хештеги: добавить тематический блок

**Что сохраняется:**
- Тон, энергия, персона
- Хук (адаптированный под 125 символов)
- Ключевые лексические маркеры
- Основной CTA (адаптированный)

---

## Универсальные правила

### Всегда сохраняется (ядро голоса)
- Лексика: характерные слова и выражения автора
- Тон: формальность, энергия, эмоциональный окрас
- Персона: как автор позиционирует себя
- Ценности: о чём говорит, чего избегает
- Ритм: соотношение коротких/длинных предложений
- Фирменные приёмы: уникальные стилистические фишки

### Всегда адаптируется (оболочка)
- Длина: под ограничения платформы
- Форматирование: bold/italic/markdown → доступные средства
- Хештеги: нет в TG, есть в VK и IG
- Ссылки: inline / plain / "ссылка в bio"
- Медиа-ожидания: TG=текст-first, IG=визуал-first
- CTA: направлять на доступные для платформы действия

### Таблица адаптации

| Элемент | Telegram | VK | Instagram |
|---------|----------|-----|-----------|
| Длина | 500-2000 сим | 500-3000 сим | 500-1000 сим |
| Markdown | ✅ полный | ❌ ограничен | ❌ нет |
| Хештеги | ❌ | 3-5 штук | 15-30 штук |
| Ссылки | inline | plain URL | только в bio |
| Эмодзи | по профилю | по профилю +10% | по профилю +30% |
| Абзацы | пустая строка | пустая строка | перенос строки |
| CTA | подписка, бот, канал | подписка, комментарий | follow, link in bio |
