# Каталог модулей — учебная программа {{SCHOOL_NAME}}

> 4 месяца, 16 занятий по 90 минут. Каждое занятие = 1 раз в неделю.
> Каноничные параметры: `projects/Synapse/SYNAPSE_CANONICAL.md`

## Месяц 1: «AI-Исследователь» (Основы)

**Цель месяца:** Ребёнок понимает, что такое AI, умеет вести диалог с текстовым AI, создаёт первые визуальные работы и собирает мини-проект.

---

### Занятие 1: Распаковка мечты

**Название для учеников:** «Что я хочу создать?»

**Цели обучения:**
1. Составить карту своих интересов и увлечений
2. Увидеть первый WOW-момент — AI создаёт что-то по твоему описанию за 30 секунд
3. Сформулировать идею проекта, который будет развиваться весь курс

**Инструменты:** ChatGPT (или Claude), Midjourney (или Leonardo.ai), Canva

**Артефакт урока:** Карта интересов (визуальная, сделанная с помощью AI) + записанная идея личного проекта

**Разминка (10 мин):**
- Знакомство: имя + «одна вещь, которую я бы попросил у волшебника»
- Преподаватель показывает: «Смотрите, я прошу AI нарисовать [берёт ответ ребёнка] — и вот результат за 30 секунд!»

**Навык (15 мин):**
- Что такое AI? (Не робот, а инструмент, который понимает текст и создаёт по описанию)
- Демо: преподаватель показывает 3 примера — текст, картинка, идея от AI
- Каждый ребёнок вводит свой первый запрос в ChatGPT: «Придумай мне 5 идей проекта про [моё увлечение]»

**Проект (40 мин):**
- Заполнение «Карты интересов»: 4 квадрата — Хобби, Школьные предметы, Мечты, Навыки
- AI помогает расширить каждый квадрат (подбрасывает идеи)
- Выбор 1 идеи для личного проекта → записать в тетрадь/Notion

**Шеринг (15 мин):**
- Каждый показывает свою карту и говорит: «Мой проект будет про...»

**Wrap-up (10 мин):**
- Итог: «Сегодня вы познакомились с AI как инструментом для создания, а не просто поиска»
- Превью: «В следующий раз — научимся общаться с AI так, чтобы он делал именно то, что мы хотим»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Карта интересов | Рисуют от руки + AI-иллюстрации | Делают в Canva или Notion |
| Первый запрос | Готовый шаблон: «Придумай 5 идей про ___» | Формулируют самостоятельно |
| Идея проекта | Преподаватель помогает сузить | Выбирают автономно |
| WOW-демо | Преподаватель показывает | Дети сами пробуют генерацию |

### Занятие 2: Мастерство текстового AI

**Название для учеников:** «Говори с AI на его языке»

**Цели обучения:**
1. Научиться писать чёткие промпты (роль + задача + контекст + формат)
2. Освоить итеративное уточнение — «не то → уточняю → лучше»
3. Проверять факты: AI может ошибаться, и это нормально

**Инструменты:** ChatGPT, Claude

**Артефакт урока:** 3 промпта (плохой → средний → отличный) по одной теме + результат «Факт-чек» (найденная ошибка AI)

**Разминка (10 мин):**
- AI-челлендж: «Попросите AI написать стихотворение про [случайное слово]. Кто получит самый смешной результат?»
- Обсуждение: почему результаты разные, если задача одна?

**Навык (15 мин):**
- Формула промпта: **Роль** (ты — ...) + **Задача** (сделай ...) + **Контекст** (для кого, зачем) + **Формат** (в виде списка / таблицы / истории)
- Демо с одним и тем же запросом:
  - Плохой: «Расскажи про космос»
  - Средний: «Расскажи про Марс для школьника 10 лет»
  - Отличный: «Ты — учёный-астроном. Объясни ребёнку 10 лет, почему Марс красный. Используй простые сравнения. Формат: 3 абзаца + 1 интересный факт»
- Практика: каждый улучшает свой промпт по формуле

**Проект (40 мин):**
- Задание 1 (20 мин): Написать 3 версии промпта по теме своего проекта → сравнить результаты
- Задание 2 (10 мин): «Факт-чек» — задать AI вопрос по теме, проверить ответ в поиске, найти ошибку
- Задание 3 (10 мин): Итеративное уточнение — взять ответ AI, написать «это хорошо, но измени X и добавь Y»

**Шеринг (15 мин):**
- Каждый показывает свой лучший промпт и результат
- Кто нашёл самую интересную ошибку AI?

**Wrap-up (10 мин):**
- Главный урок: «AI — не оракул, а инструмент. Чем точнее вопрос — тем лучше ответ»
- Превью: «В следующий раз — создаём картинки! AI будет рисовать по вашим описаниям»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Формула промпта | Упрощённая: «Кто ты + Что сделай + Для кого» | Полная: Роль + Задача + Контекст + Формат |
| Факт-чек | Преподаватель помогает проверить | Проверяют самостоятельно в 2+ источниках |
| Итерация | 2 версии промпта | 3 версии + рефлексия «что изменилось» |
| Инструмент | Только ChatGPT | ChatGPT + Claude (сравнить ответы) |

### Занятие 3: Визуальный AI

**Название для учеников:** «AI рисует по твоим словам»

**Цели обучения:**
1. Создать изображение через AI по текстовому описанию
2. Понять принципы визуального промпта: стиль, композиция, детали
3. Достичь стилистической консистентности (серия из 2–3 картинок в одном стиле)

**Инструменты:** Leonardo.ai (основной), Midjourney (если доступен), Canva

**Артефакт урока:** Персонаж для своего проекта (3 картинки в одном стиле) — аватар, сцена, эмоция

**Разминка (10 мин):**
- Игра «Угадай промпт»: преподаватель показывает 3 AI-картинки, дети пытаются угадать, что было написано в промпте
- Обсуждение: «Что нужно указать, чтобы AI нарисовал ТОЧНО то, что в голове?»

**Навык (15 мин):**
- Анатомия визуального промпта:
  - **Объект** (что): a cute robot cat
  - **Стиль** (как): in Pixar 3D style / watercolor / pixel art
  - **Детали** (какой): wearing a red scarf, smiling, big eyes
  - **Фон/сцена** (где): in a cozy library, warm lighting
  - **Технические параметры**: high quality, 4K, detailed
- Демо: один и тот же персонаж в 3 стилях
- Понятие «стилистическая консистентность»: как делать серию в одном стиле (seed, стиль-референс)

**Проект (40 мин):**
- Создание персонажа для своего проекта:
  1. Описать персонажа текстом (5 мин)
  2. Сгенерировать первую версию → обсудить с преподавателем (10 мин)
  3. Уточнить промпт → сгенерировать 2-ю версию (10 мин)
  4. Создать серию: персонаж в 3 разных ситуациях, одинаковый стиль (15 мин)

**Шеринг (15 мин):**
- Каждый показывает своего персонажа (3 картинки)
- Голосование: «Чей персонаж самый запоминающийся?»

**Wrap-up (10 мин):**
- Главный урок: «Визуальный AI — это не магия, а навык точного описания. Чем лучше опишешь — тем круче результат»
- Превью: «Следующее занятие — собираем первый мини-проект, где текст и картинки работают вместе!»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Инструмент | Leonardo.ai (простой интерфейс) | Leonardo.ai + Midjourney (если доступен) |
| Промпт | Готовые шаблоны с пропусками: «A ___ in ___ style, ___ colors» | Пишут с нуля, экспериментируют с параметрами |
| Серия | 2 картинки | 3 картинки + осознанная работа со стилем |
| Консистентность | Преподаватель помогает подобрать стиль | Сами работают с seed/стиль-референсами |

### Занятие 4: Первый мини-проект

**Название для учеников:** «Собираем всё вместе»

**Цели обучения:**
1. Объединить текстовый и визуальный AI в один цельный проект
2. Оформить результат для показа (Canva-постер / презентация / страница)
3. Презентовать свой проект группе и получить обратную связь

**Инструменты:** ChatGPT / Claude, Leonardo.ai, Canva

**Артефакт урока:** Готовый мини-проект в одном из форматов: постер, мини-книга (4 страницы), презентация (5 слайдов), Telegram-пост

**Разминка (10 мин):**
- Быстрый обзор: «Что мы умеем за 3 занятия?» — промпты, картинки, идея проекта
- Преподаватель показывает пример готового мини-проекта (эталон)

**Навык (15 мин):**
- Сборка проекта: как соединить текст и визуал
- Композиция в Canva: заголовок + картинка + текст = страница
- 3 формата на выбор:
  - **Постер** (1 страница): заголовок + большая картинка + 3 факта
  - **Мини-книга** (4 страницы): обложка + 2 разворота + итог
  - **Презентация** (5 слайдов): титул + проблема + решение + картинки + вывод

**Проект (40 мин):**
- Каждый ребёнок выбирает формат и собирает свой проект:
  1. Генерация текста через ChatGPT/Claude (10 мин)
  2. Генерация/доработка визуала (10 мин)
  3. Сборка в Canva (15 мин)
  4. Финальные штрихи, проверка (5 мин)

**Шеринг (15 мин):**
- Мини-презентации: каждый показывает проект (2 мин на человека)
- Обратная связь от группы: «Мне понравилось... / Я бы добавил...»

**Wrap-up (10 мин):**
- Итоги первого месяца: «Вы прошли путь от "что такое AI" до готового проекта за 4 занятия»
- Сертификат уровня «AI-Исследователь» (можно напечатать или показать на экране)
- Превью 2 месяца: «Дальше — круче. Истории, видео, свой канал!»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Формат | Постер (проще всего) или мини-книга | Презентация или мини-книга |
| Canva | Готовый шаблон, меняют текст и картинки | С нуля или сложный шаблон |
| Презентация | 1 мин, можно с помощью преподавателя | 2 мин, самостоятельно, отвечают на вопросы |
| Обратная связь | Только позитив: «Что понравилось?» | Позитив + конструктив: «Что бы улучшил?» |

## Месяц 2: «AI-Создатель» (Созидание)

**Цель месяца:** Ребёнок создаёт комплексные проекты с AI — истории, видео, аудио. Запускает свой канал/блог и учится планировать контент.

---

### Занятие 5: AI-Сторителлинг

**Название для учеников:** «Создай историю, которую невозможно забыть»

**Цели обучения:**
1. Понять структуру истории (герой → проблема → путь → решение)
2. Создать историю / комикс с помощью AI (текст + визуал)
3. Освоить технику «AI как соавтор» — не AI пишет за тебя, а ты направляешь

**Инструменты:** ChatGPT / Claude, Leonardo.ai, Canva

**Артефакт урока:** Комикс (4–6 кадров) или иллюстрированный рассказ (2–3 страницы)

**Разминка (10 мин):**
- Игра «Завяжи историю»: преподаватель говорит начало, каждый ребёнок добавляет 1 предложение
- Вопрос: «Как AI мог бы помочь написать историю лучше?»

**Навык (15 мин):**
- Структура истории: Герой → Мир → Проблема → Путь → Кульминация → Развязка
- Техника «AI-соавтор»: пишешь план → AI разворачивает → ты редактируешь → AI иллюстрирует
- Демо: преподаватель создаёт мини-историю за 5 минут (текст + 2 картинки)

**Проект (40 мин):**
1. План истории: 6 пунктов по структуре (10 мин)
2. AI разворачивает каждый пункт в абзац (10 мин)
3. Генерация иллюстраций к ключевым сценам (10 мин)
4. Сборка в формат комикса или книги (10 мин)

**Шеринг (15 мин):**
- Чтение/показ историй
- Обсуждение: «Где AI помог больше всего?»

**Wrap-up (10 мин):**
- Главный урок: «AI — лучший соавтор, но режиссёр — ты»
- Превью: «Следующее занятие — добавим видео и звук!»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Формат | Комикс (4 кадра, меньше текста) | Иллюстрированный рассказ (2–3 стр.) |
| Структура | Упрощённая: Герой → Проблема → Решение | Полная: 6 элементов |
| Промпт | «Напиши историю про [персонажа], который [проблема]» | «Ты — сценарист. Напиши историю по плану: ...» |
| Редактура | Преподаватель помогает улучшить | Сами редактируют + peer review |

### Занятие 6: AI-Видео и Аудио

**Название для учеников:** «Оживляем картинки и добавляем голос»

**Цели обучения:**
1. Создать короткое видео из AI-картинок (анимация изображения)
2. Сгенерировать музыку или озвучку через AI
3. Понять концепцию раскадровки (storyboard)

**Инструменты:** Runway (или Pika), Suno (музыка), ElevenLabs (голос, демо), Canva (раскадровка)

**Артефакт урока:** 15–30 секундное видео с AI-анимацией + музыкой / озвучкой

**Разминка (10 мин):**
- Преподаватель показывает 3 AI-видео (Runway, Pika) — дети угадывают, что реальное, что AI
- Обсуждение: «Как вы думаете, сколько времени это заняло?»

**Навык (15 мин):**
- Раскадровка: 4–6 кадров → что происходит → какой текст → какая музыка
- Демо Runway: загрузка картинки → анимация → результат за 1 мин
- Демо Suno: текстовое описание → музыка за 30 сек
- Принцип: «Сначала план → потом генерация»

**Проект (40 мин):**
1. Нарисовать раскадровку на бумаге / в Canva (10 мин)
2. Сгенерировать/выбрать 3–4 ключевых кадра в Leonardo.ai (10 мин)
3. Анимировать 1–2 кадра в Runway/Pika (10 мин)
4. Добавить музыку (Suno) или озвучку (ElevenLabs демо) (10 мин)

**Шеринг (15 мин):**
- Просмотр видео каждого ученика
- Голосование: «Лучшая музыка», «Лучшая анимация», «Лучшая история»

**Wrap-up (10 мин):**
- Главный урок: «Видео и аудио — это тоже AI-инструменты, и вы уже умеете их использовать»
- Превью: «В следующий раз — запускаем свой канал!»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Видео | 1 анимированный кадр + музыка | 2–3 кадра, склейка, озвучка |
| Инструменты | Runway (проще) | Runway + Pika (сравнить) |
| Раскадровка | 4 кадра на бумаге | 6 кадров в Canva с описаниями |
| Музыка | Suno по описанию жанра | Suno с детальным описанием настроения |

### Занятие 7: Запуск канала

**Название для учеников:** «Твой первый медиа-проект»

**Цели обучения:**
1. Создать и оформить Telegram-канал (или блог)
2. Составить контент-план на 2 недели (4–6 постов)
3. Подготовить и опубликовать первый пост с AI-контентом

**Инструменты:** Telegram, ChatGPT / Claude, Canva, Leonardo.ai

**Артефакт урока:** Оформленный Telegram-канал + первый опубликованный пост + контент-план на 2 недели

**Разминка (10 мин):**
- Обзор каналов/блогов, которые ведут дети и подростки (примеры)
- Вопрос: «Если бы у тебя был канал — о чём бы ты писал?»

**Навык (15 мин):**
- Как оформить канал: название, аватар (AI-генерация), описание
- Контент-план: что публиковать, когда, зачем
- Формула поста: Заголовок-крючок + Основная мысль + Картинка + Призыв к действию
- Демо: создание поста за 5 минут с AI

**Проект (40 мин):**
1. Создать Telegram-канал: название, аватар через AI, описание (10 мин)
2. Составить контент-план: 4–6 тем + даты (10 мин)
3. Создать первый пост: текст (ChatGPT) + картинка (Leonardo.ai) + оформление (10 мин)
4. Опубликовать и поделиться ссылкой с группой (10 мин)

**Шеринг (15 мин):**
- Каждый показывает свой канал и первый пост
- Группа подписывается друг на друга

**Wrap-up (10 мин):**
- Задание на неделю: опубликовать ещё 1–2 поста по контент-плану
- Превью: «Следующее занятие — проверяем, что получилось, и улучшаем!»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Платформа | Telegram-канал (с помощью преподавателя) | Telegram-канал самостоятельно (или блог) |
| Контент-план | 4 поста, простые темы | 6 постов, разнообразные форматы |
| Первый пост | По шаблону: «Привет! Я [имя], и мой канал про...» | Свой стиль, с крючком и CTA |
| Безопасность | Преподаватель объясняет: НЕ публиковать личные данные | Обсуждение: приватность, цифровой след |

### Занятие 8: Контрольная точка проектов

**Название для учеников:** «Покажи, чему научился — и стань лучше»

**Цели обучения:**
1. Представить все проекты за 2 месяца в структурированном виде
2. Дать и получить конструктивную обратную связь (peer review)
3. Определить 1–2 проекта для углубления в месяце 3

**Инструменты:** Все ранее использованные, Canva (для презентации)

**Артефакт урока:** Портфолио-презентация (5–7 слайдов) + план улучшений на основе обратной связи

**Разминка (10 мин):**
- Рефлексия: «Что было самым сложным? Что далось легче всего?»
- Быстрый обзор: преподаватель показывает таймлайн — что создали за 8 занятий

**Навык (15 мин):**
- Как давать обратную связь: 2 звезды + 1 пожелание (2 stars & 1 wish)
- Как принимать обратную связь: не защищаться, а записывать
- Как создать мини-портфолио: титульный слайд + по слайду на проект + итоговый слайд

**Проект (40 мин):**
1. Собрать портфолио-презентацию из всех проектов (15 мин)
2. Мини-презентации + peer review (20 мин — по 3 мин на человека)
3. Записать план улучшений: что доработать, что добавить (5 мин)

**Шеринг (15 мин):**
- Уже включён в проектную часть (peer review)

**Wrap-up (10 мин):**
- Сертификат уровня «AI-Создатель»
- Итоги: «За 2 месяца вы научились работать с текстом, картинками, видео, аудио и запустили свои каналы»
- Превью 3 месяца: «Теперь — системы. Как связать AI-инструменты между собой и автоматизировать»

**Адаптация по возрасту:**

| Портфолио | 5 слайдов, преподаватель помогает | 7 слайдов, самостоятельно |
| Peer review | Только позитив (2 звезды) | 2 звезды + 1 пожелание |
| План улучшений | Устно, 1 пункт | Письменно, 2–3 пункта |
| Презентация | 2 мин, можно читать | 3 мин, рассказывать своими словами |

### Занятие 9: Мульти-инструментные цепочки

**Название для учеников:** «AI-конвейер: когда инструменты работают вместе»

**Цели обучения:**
1. Понять концепцию workflow — цепочка инструментов для одной задачи
2. Создать простой конвейер: текст → картинка → видео → пост
3. Познакомиться с автоматизацией (Make.com / Zapier — обзорно)

**Инструменты:** ChatGPT / Claude, Leonardo.ai, Runway, Canva, Make.com (демо)

**Артефакт урока:** Один пост/видео, созданный через цепочку из 3+ инструментов + схема workflow

**Разминка (10 мин):**
- Аналогия: «Конвейер на заводе — каждый станок делает своё. AI-инструменты — то же самое»
- Вопрос: «Какие инструменты мы уже знаем? Как они могут работать вместе?»

**Навык (15 мин):**
- Концепция workflow: вход → обработка → выход → вход следующего инструмента
- Демо цепочки: Тема → ChatGPT (текст) → Leonardo.ai (картинка) → Runway (анимация) → Canva (пост)
- Обзор Make.com: «Есть инструменты, которые соединяют AI автоматически» (показать интерфейс, не углубляться)

**Проект (40 мин):**
1. Выбрать задачу: пост для канала / видео-открытка / мини-реклама (5 мин)
2. Нарисовать схему workflow: какой инструмент → что делает → куда передаёт (10 мин)
3. Выполнить цепочку: создать контент шаг за шагом (20 мин)
4. Оформить и опубликовать (5 мин)

**Шеринг (15 мин):**
- Показать результат + схему workflow
- Обсуждение: «Чья цепочка самая длинная? Самая эффективная?»

**Wrap-up (10 мин):**
- Главный урок: «Мощь AI — не в одном инструменте, а в умении соединять их»
- Превью: «Следующее занятие — серьёзный разговор: как AI может обманывать и как не попасться»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Workflow | 3 инструмента, преподаватель ведёт | 4+ инструмента, самостоятельно |
| Схема | Рисуют стрелками на бумаге | Делают в Canva / draw.io |
| Make.com | Только смотрят демо | Могут попробовать создать простой сценарий |
| Задача | Пост для канала | Мини-реклама или видео-проект |

### Занятие 10: AI-Этика и критическое мышление

**Название для учеников:** «AI может врать. Как не попасться?»

**Цели обучения:**
1. Распознавать дипфейки (видео, аудио, фото)
2. Понимать предвзятость AI (bias) — почему AI иногда «думает» стереотипами
3. Освоить методы верификации: обратный поиск, перекрёстная проверка, здравый смысл

**Инструменты:** Google Reverse Image Search, ChatGPT (для демо ошибок), примеры дипфейков (заранее подготовленные)

**Артефакт урока:** «Карточка детектива» — памятка из 5 методов проверки AI-контента

**Разминка (10 мин):**
- Игра «Реальное или AI?»: 10 картинок/текстов — угадать, что создано AI
- Подсчёт: сколько угадали? (обычно 50–60% — показывает, что отличить сложно)

**Навык (15 мин):**
- Что такое дипфейк: видео, аудио, фото — примеры (безопасные, обучающие)
- Что такое bias: «Попросите AI нарисовать "учёного" — кого он нарисует? А "медсестру"?» (демо)
- 5 методов проверки:
  1. Обратный поиск изображений (Google)
  2. Проверка фактов в 2+ источниках
  3. Поиск артефактов на AI-картинках (пальцы, текст, фон)
  4. Вопрос «Кому выгодно?» (для фейковых новостей)
  5. Здравый смысл: «Звучит слишком хорошо/плохо, чтобы быть правдой?»

**Проект (40 мин):**
1. Квест «AI-детектив»: 5 заданий на проверку (20 мин):
   - Найти ошибку в AI-тексте
   - Найти артефакт на AI-картинке
   - Проверить «факт» из ChatGPT
   - Определить bias в результатах генерации
   - Оценить фейковую новость
2. Создать «Карточку детектива» — памятку с 5 методами проверки (10 мин)
3. Обмен карточками + дополнение (10 мин)

**Шеринг (15 мин):**
- Обсуждение: «Что удивило больше всего?»
- Каждый озвучивает свой главный вывод

**Wrap-up (10 мин):**
- Главный урок: «AI — мощный инструмент, но ответственность — на нас. Проверяй, думай, не верь слепо»
- Превью: «Следующее занятие — продвинутый промптинг. Научимся "программировать" поведение AI»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Дипфейки | Упрощённо: «AI может рисовать лица, которых нет» | Подробно: видео, аудио, социальные последствия |
| Bias | «AI иногда рисует стереотипно — это не правда» | Системный bias, датасеты, как его уменьшить |
| Квест | 3 задания попроще | 5 заданий, включая оценку новости |
| Карточка | 3 метода | 5 методов + примеры |

### Занятие 11: Продвинутый промптинг

**Название для учеников:** «Программируй AI без кода»

**Цели обучения:**
1. Написать системный промпт (задать AI постоянную роль и правила)
2. Создать Custom GPT или кастомный чат с инструкцией
3. Понять концепцию AI-агента: «AI, который выполняет задачу по шагам»

**Инструменты:** ChatGPT (Custom GPTs), Claude (Projects/System prompt)

**Артефакт урока:** Свой Custom GPT (или кастомный чат в Claude) — AI-помощник под свою задачу

**Разминка (10 мин):**
- Вопрос: «Представь, что у тебя есть личный AI-помощник, который знает всё про твоё хобби. Каким бы он был?»
- Преподаватель показывает пример Custom GPT: «Это AI, который всегда отвечает как пират-учёный»

**Навык (15 мин):**
- Что такое системный промпт: «Инструкция для AI, которую он помнит всегда»
- Структура:
  - **Роль:** «Ты — [кто]»
  - **Правила:** «Всегда [делай X]. Никогда [не делай Y]»
  - **Стиль:** «Отвечай [как]»
  - **Ограничения:** «Только по теме [X]»
- Демо: создание Custom GPT за 3 минуты
- Концепция агента: «AI + инструкция + инструменты = AI-агент, который решает задачи»

**Проект (40 мин):**
1. Придумать идею AI-помощника (5 мин):
   - Помощник по домашке / Тренер по хобби / Генератор идей / Персональный стилист и т.д.
2. Написать системный промпт (15 мин):
   - Роль, правила, стиль, ограничения
3. Создать Custom GPT или кастомный чат (10 мин)
4. Протестировать: задать 5 вопросов, проверить, что AI следует инструкциям (10 мин)

**Шеринг (15 мин):**
- Каждый демонстрирует своего AI-помощника
- Группа задаёт вопросы: «А если спросить его про...?»

**Wrap-up (10 мин):**
- Главный урок: «Вы только что "запрограммировали" AI без единой строчки кода — промптами»
- Превью: «Следующее занятие — берём лучшие проекты и доводим до блеска»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Системный промпт | 3–5 строк, простые правила | 10–15 строк, сложные правила + примеры |
| Custom GPT | Создают вместе с преподавателем | Создают самостоятельно |
| Агент | «AI, который помогает по шагам» (концепция) | Углублённо: инструменты, контекст, память |
| Тестирование | 3 вопроса | 5 вопросов + попытка «сломать» промпт |

**Инструменты:** Все ранее использованные (по выбору ученика)

**Артефакт урока:** Обновлённая версия проекта с документацией изменений: «Было → Стало → Почему»

**Проект (40 мин):**

4. Документация: «Было → Стало → Почему» (5 мин)

**Шеринг (15 мин):**
- Каждый показывает «до и после»
- Голосование: «Самая крутая трансформация»

**Wrap-up (10 мин):**
- Сертификат уровня «AI-Архитектор»
- Главный урок: «Профессионалы не делают идеально с первого раза — они итерируют»
- Превью 4 месяца: «Последний рывок — портфолио, презентация, демо-день для родителей!»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Количество проектов | 1 проект | 1–2 проекта |
| Документация | Устно: «Я поменял X, потому что Y» | Письменно: таблица «Было → Стало → Почему» |
| Чеклист | 3 критерия | 5 критериев |
| Обратная связь | Преподаватель направляет | Peer review + самооценка |

**Wrap-up (10 мин):**

### Занятие 14: Навыки презентации

**Название для учеников:** «Расскажи — и тебя запомнят»

**Цели обучения:**
1. Структурировать питч: Проблема → Решение → Демо → Вывод (ПРДВ)
2. Отрепетировать выступление с таймером (3 мин)
3. Использовать AI для подготовки к выступлению (генерация тезисов, тренировка)

**Инструменты:** ChatGPT / Claude (генерация тезисов, вопросы-ответы), таймер, камера (для записи репетиции)

**Артефакт урока:** Готовый питч (3 мин) + презентация (3–5 слайдов) + записанная видео-репетиция

**Разминка (10 мин):**
- Видео: пример хорошей детской презентации (TED-Ed или аналог)
- Обсуждение: «Что делает выступление интересным?»

**Навык (15 мин):**
- Структура питча ПРДВ:
  - **Проблема:** «Знаете ли вы, что...» (крючок)
  - **Решение:** «Я создал...» (что сделал)
  - **Демо:** «Давайте покажу...» (живая демонстрация)
  - **Вывод:** «Теперь я умею...» (чему научился)
- Правила выступления: контакт глазами, паузы, громкость, жесты
- Лайфхак: «Попроси AI сгенерировать 5 вопросов, которые могут задать, и подготовь ответы»

**Проект (40 мин):**
1. Написать тезисы питча по структуре ПРДВ (10 мин)
2. Подготовить 3–5 слайдов (10 мин)
3. Репетиция с таймером — 2 прогона (15 мин)
4. Запись видео-репетиции (5 мин)

**Шеринг (15 мин):**
- Каждый выступает перед группой (3 мин) — генеральная репетиция
- Обратная связь от преподавателя и группы

**Wrap-up (10 мин):**
- Главный урок: «Умение рассказать о своём проекте — не менее важно, чем умение его создать»
- Превью: «Осталось 2 занятия: финальная шлифовка + демо-день с родителями!»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Питч | 2 мин, можно с карточками-подсказками | 3 мин, без карточек |
| Слайды | 3 слайда | 5 слайдов |
| AI-подготовка | AI генерирует тезисы, ребёнок пересказывает | Ребёнок пишет → AI улучшает → ребёнок финализирует |
| Вопросы | Преподаватель задаёт 1 вопрос | Группа задаёт 2–3 вопроса |

### Занятие 15: Финальная шлифовка

**Название для учеников:** «Последний рывок перед стартом»

**Цели обучения:**
1. Довести проект до презентационного качества
2. Подготовить всё для демо-дня: портфолио, презентация, демо
3. Помочь друг другу: peer review + совместная работа

**Инструменты:** Все ранее использованные (по выбору)

**Артефакт урока:** Финальная версия проекта, готовая к демо-дню

**Разминка (10 мин):**
- Энерджайзер: «Через 1 занятие — ваши родители увидят, чему вы научились. Как себя чувствуете?»
- Составление чеклиста: что нужно доделать каждому

**Навык (15 мин):**
- «Последняя миля»: 80% работы занимает 20% времени, последние 20% — 80% времени
- Чеклист финального качества:
  - [ ] Проект работает / выглядит как задумано
  - [ ] Портфолио обновлено
  - [ ] Презентация отрепетирована
  - [ ] Демо подготовлено (знаю, что показывать)
  - [ ] Нет опечаток, сломанных картинок, пустых страниц

**Проект (40 мин):**
- Индивидуальная работа по чеклисту
- Преподаватель — в режиме ментора: подходит к каждому, помогает
- Peer-помощь: если закончил раньше — помоги соседу

**Шеринг (15 мин):**
- Финальный прогон питча (по 2 мин)
- Группа оценивает по 3 критериям: содержание, подача, визуал

**Wrap-up (10 мин):**
- Проверка готовности: «Все готовы? Портфолио? Презентация? Демо?»
- Организационные вопросы: время демо-дня, что принести, кто из родителей придёт
- Мотивация: «Вы прошли огромный путь. Завтра — показываете результат!»

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Чеклист | 3 пункта | 5 пунктов |
| Помощь | Преподаватель активно помогает | Peer-менторство: старшие помогают друг другу |
| Прогон | 1 раз, с подсказками | 2 раза, без подсказок |
| Фокус | Чтобы работало и выглядело хорошо | Чтобы работало + было убедительно |

### Занятие 16: Демо-день

**Название для учеников:** «Добро пожаловать в будущее!»

**Цели обучения:**
1. Представить свой проект аудитории (родители, друзья, другие ученики)
2. Ответить на вопросы аудитории
3. Отрефлексировать свой путь за 4 месяца

**Инструменты:** Проектор, портфолио учеников, сертификаты (заранее подготовленные)

**Артефакт урока:** Публичная презентация + сертификат выпускника + фото/видео на память

**Структура демо-дня (90 мин):**

```
0:00–0:10  Вступление преподавателя
           — Что такое {{SCHOOL_NAME}}, чему мы учим
           — Обзор программы: 4 месяца, 16 занятий, X проектов
           — Представление учеников

0:10–0:55  Презентации учеников
           — По 5–7 мин на ученика (зависит от размера группы)
           — Формат: питч (3 мин) + демо (2 мин) + вопросы (1–2 мин)

0:55–1:10  Интерактивная часть
           — Родители пробуют AI-инструменты вместе с детьми
           — Дети показывают родителям, как пользоваться ChatGPT / генерацией картинок
           — «Мастер-класс наоборот»: ребёнок учит взрослого

1:10–1:20  Вручение сертификатов
           — Вызывать каждого по имени
           — Кратко: главное достижение каждого ученика
           — Сертификат «Выпускник {{SCHOOL_NAME}}: AI-мышление»

1:20–1:30  Фото, общение, фидбэк
           — Общее фото группы
           — Анкета для родителей (обратная связь)
           — Информация о следующих программах (продвинутый курс, индивидуальные занятия)
```

**Адаптация по возрасту:**

| Аспект | Младшая (9–11) | Старшая (12–14) |
|--------|---------------|-----------------|
| Презентация | 4 мин (питч 2 мин + демо 2 мин) | 6 мин (питч 3 мин + демо 2 мин + вопросы 1 мин) |
| Вопросы | Преподаватель модерирует | Ребёнок отвечает самостоятельно |
| Интерактив | Ребёнок показывает 1 инструмент | Ребёнок проводит мини-мастеркласс |
| Сертификат | «AI-Исследователь» | «AI-Создатель» (или «AI-Архитектор» для продвинутых) |
