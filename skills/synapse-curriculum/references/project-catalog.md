# Каталог проектов {{SCHOOL_NAME}}

> 35 проектных идей для учеников 9–14 лет. Каждый проект можно выдать как задание на занятии, как домашний проект или как вариант для демо-дня.
> Организация: по направлению (Творческий / Технический / Бизнес), сложности и возрасту.

---

## Как пользоваться каталогом

1. Определить интересы ученика (из «Карты интересов» Занятия 1)
2. Выбрать направление: Творческий / Технический / Бизнес
3. Проверить возрастную группу и сложность
4. Проверить инструменты в `tool-matrix.md`
5. Адаптировать под конкретного ученика

**Сложность:**
- 🟢 1 занятие (90 мин) — можно сделать за одно занятие
- 🟡 2–3 занятия — нужно несколько занятий
- 🔴 4+ занятий — долгосрочный проект (для финального проекта)
