### HARD GATES (обязательные чекпоинты)
- **Шаги 1-5 (Фаза 1) ОБЯЗАТЕЛЬНЫ перед Фазой 2** — без концепта и распаковки прогрев будет пустым
- **Шаг 6 (распаковка) должен дать документ 2000+ слов** — иначе не хватит материала для 14-21 дня контента
- **Шаг 12 (прогрев) минимум 14 дней** — запуск без прогрева = провал продаж
- **Запуск без декомпозиции = FAIL** — нельзя стартовать без цифр (сколько лидов нужно?)

# launch-system — Оркестратор запуска инфопродукта

## Purpose

Полный цикл запуска инфопродукта в Telegram: от первой встречи с экспертом до ретроспективы после продаж. 17-шаговая система на основе фреймворков Тимочко и Пырикова. Это мета-скилл (оркестратор) — он координирует вызов других скиллов на каждом этапе.

## When to Use

- Пользователь говорит: "запуск", "запускаю курс", "план запуска", "система запусков", "помоги с запуском", "этапы запуска", "запуск в Telegram"
- Нужно спланировать timeline запуска продукта
- Нужна декомпозиция цели по выручке в конкретные шаги
- Работа с экспертом: от распаковки до продаж

## 4 столпа запуска (Пыриков)

Каждый запуск стоит на 4 столпах. Если один слабый — запуск проседает:

1. **Воронки** — путь от первого касания до покупки (лестница Ханта)
2. **Контент** — Reels, сторис, посты, прогрев (70+ идей из Конструктора прогревов)
3. **Трафик** — откуда приходят люди (TG Ads, посевы, кросс-промо, Reels)
4. **Команда** — с шага 10+ нужна команда: продюсер, SMM, отдел продаж, дизайнер

## Декомпозиция запуска

Для каждого запуска считать обратно от цели:

```
Цель выручки: [X]₽
÷ Средний чек: [Y]₽
= Количество продаж: [N]
÷ CR продажи: [A]% (обычно 3-10%)
= Количество заявок: [M]
÷ CR заявки: [B]% (обычно 10-30%)
= Охват (регистрации на бесплатник): [K]
× CPL (стоимость лида): [C]₽
= Бюджет на трафик: [Z]₽
```

**Правило Пырикова:** бюджет на трафик = 30-40% от ожидаемой выручки.

## Трафик в Telegram (Пыриков)

| Канал | Описание | Особенности |
|-------|----------|-------------|
| **TG Ads** | Таргетинг на подписчиков конкурентов | Самый контролируемый |
| **Посевы** | Реклама в тематических каналах | Быстрый охват |
| **Кросс-промо** | Взаимный пиар с каналами | Бесплатно, но требует переговоров |
| **Reels -> TG** | Cross-platform: Instagram/YouTube Shorts -> ссылка в Telegram | Масштабируется органикой |

## Метрики запуска (Пыриков)

| Метрика | Бенчмарк |
|---------|----------|
| Churn подписчиков | 50-60%/год |
| Спайк последнего дня продаж | x5-x7 от среднего дня |
| Доля рассрочки | 50-90% оплат |
| Бюджет на трафик | 30-40% от ожидаемой выручки |
| CR отдел продаж | до 60% от общей выручки |

1. Спросить: дата продаж, продукт, цена, цель выручки
2. Посчитать декомпозицию (выручка -> продажи -> заявки -> охват -> бюджет)

4. Для каждого шага указать: конкретную дату, задачу, ответственного, суб-скилл

## Anti-patterns

| Антипаттерн | Почему плохо | Что делать |
|-------------|-------------|-----------|
| Запуск без прогрева | Продажи с нуля = низкий CR | Минимум 10 дней прогрева (шаги 11-12) |
| Прогрев > 30 дней | Аудитория устаёт, охваты падают | Держать прогрев 10-25 дней |
| Нет декомпозиции | Действия без цифр = гадание | Всегда считать: выручка -> продажи -> заявки -> охват -> бюджет |
| Всё делать самому | Выгорание, упущенные задачи | С шага 10+ нужна команда |
| Нет дедлайна продаж | Без дедлайна нет спайка (x5-x7) | Жёсткое окно продаж: 5-7 дней |
| Одинаковый прогрев | "Прогревная слепота" — аудитория не реагирует | Использовать инфоповоды (30+ идей из Тимочко) |
| Пропуск распаковки | Контент без глубины, шаблонный | Полная распаковка (60+ вопросов, 4 блока, 2 часа) |
| Игнорировать отдел продаж | Потеря до 60% возможной выручки | Подключить ОП с шага 10 |

## Чеклист поверхностного опроса (Шаг 1)

11 вопросов для первой встречи с экспертом:

1. Что продаем? Какой продукт?
2. Какие цены?
3. Почему именно такой продукт?
4. Были ли уже запуски? Какие результаты?
5. Кто аудитория? Описание сегментов
6. Какие охваты сейчас?
7. Какие каналы коммуникации (Instagram, Telegram, YouTube)?
8. Есть ли база подписчиков? Сколько?
9. Бюджет на трафик?
10. Кто в команде?
11. Когда хотите стартовать?

## Концепт продукта — 8 пунктов (Шаг 5)

1. **Для кого** — 3-4 сегмента ЦА (сегмент + описание + результат)
2. **Большое обещание** — "[Результат] за [срок] без [болей]"
3. **По итогу** — конкретные артефакты/результаты
4. **Программа** — по модулям с результатом после каждого
5. **Формат** — онлайн/оффлайн, длительность, периодичность, поддержка
6. **Тарифы** — 2-3 уровня (база / оптимум / премиум)
7. **Фишки "вау-эффекта"** — 5-10 фишек
8. **Болеутолители** — снятие барьеров ("не придется...", "без...")
