# Калибровочные примеры — Visual Replication

## Зачем

Claude склонен завышать оценку визуального сходства. Эти примеры задают рамку: что такое "хорошо" и что такое "плохо".

## Пример 1 — Excellent (92%)

**Ситуация:** Референс — тёмный лендинг с фиолетовым акцентом. Результат идентичен, единственное отличие — шрифт Inter вместо кастомного (не установлен на сервере).

**Почему 92%:** Pixel diff показывает отличия в кернинге и line-height из-за другого шрифта. Визуально — практически неотличимо.

**Вердикт:** Принять. Шрифтовые отличия неизбежны если нет точного font-файла.

## Пример 2 — Good (87%)

**Ситуация:** Референс — карточки товаров 3 в ряд. Результат: карточки есть, цвета совпадают, но spacing между карточками 24px вместо 32px, border-radius 8px вместо 12px.

**Почему 87%:** Цвета идентичны (из analyzer), но spacing и radius "на глаз". 

**Вердикт:** Принять, но лучше подтянуть spacing — gap виден при сравнении бок о бок.

## Пример 3 — Needs Work (72%)

**Ситуация:** Референс — hero-секция с градиентом. Результат: layout правильный, но градиент linear вместо radial, тень у кнопки отсутствует, hero-изображение масштабировано неправильно (contain вместо cover).

**Почему 72%:** 3 заметных отличия, каждое влияет на "ощущение". При A/B-сравнении разница очевидна.

**Вердикт:** Итерировать. Исправить: 1) градиент radial, 2) box-shadow у кнопки, 3) object-fit: cover.

## Пример 4 — Major Differences (48%)

**Ситуация:** Референс — desktop 1440px. Скриншот сделан с viewport 375px (мобильный). Весь layout переломан, карточки в столбец вместо ряда.

**Почему 48%:** Это не "плохая реализация" — это неверный viewport. Контент тот же, но layout полностью другой.

**Вердикт:** НЕ итерировать. Разобраться: viewport скриншота должен совпадать с референсом. Пересделать скриншот с правильными размерами.

## Правило для Claude

При самооценке сходства вычитай 5-10% от интуитивной оценки. Если кажется что "90%" — скорее всего 82%. Используй `visual-analyzer.py --compare` для объективной оценки, не доверяй собственному восприятию.
