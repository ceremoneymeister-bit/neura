## 1. Цветовая палитра

### Основные цвета

| Название | HEX | RGB | Роль |
|----------|-----|-----|------|
| **Acid Lime** | `#CCFF00` | 204, 255, 0 | Основной акцент. Кнопки, подсветки, CTA, выделения |
| **Pure Black** | `#000000` | 0, 0, 0 | Фон hero-секций, текст, тени, рамки |
| **Deep Purple** | `#5D3FD3` | 93, 63, 211 | Вторичный акцент. Иконки, бейджи, графики, ховеры |

### Нейтральные цвета

| Название | HEX | RGB | Роль |
|----------|-----|-----|------|
| **White** | `#FFFFFF` | 255, 255, 255 | Основной фон страницы, текст на тёмном фоне |
| **Light Grey** | `#F5F5F5` | 245, 245, 245 | Фон карточек, секций, контейнеров |
| **Dark Grey** | `#111111` | 17, 17, 17 | Футер, тёмные секции |

### Производные цвета

| Название | HEX | Роль |
|----------|-----|------|
| **Lime Hover** | `#B3E600` | Состояние hover для lime-элементов |
| **Lime 20%** | `rgba(204, 255, 0, 0.2)` | Фоновые подложки, свечение |
| **Purple 15%** | `rgba(93, 63, 211, 0.15)` | Лёгкие фоновые акценты |

### Правила использования цветов

- **Acid Lime `#CCFF00`** — ТОЛЬКО на чёрном или тёмном фоне. На белом фоне выглядит блёкло — использовать чёрную рамку вокруг
- **Deep Purple `#5D3FD3`** — вспомогательный, НЕ основной. Максимум 15-20% площади
- **Чёрный + Lime** — основная связка бренда. Должна присутствовать на каждом материале
- Никогда НЕ использовать: красный, оранжевый, розовый, пастельные тона

## 2. Типографика

### Шрифтовая система

| Роль | Шрифт | Вес | Применение |
|------|-------|-----|-----------|
| **Заголовки** | Space Grotesk | 700–800 | H1–H6, навигация, кнопки |
| **Основной текст** | Inter | 300–600 | Параграфы, подписи, списки |
| **Акцентный** | Unbounded | 400–900 | Логотип, цифры, wow-элементы, бейджи |

### Иерархия размеров

| Элемент | Desktop | Mobile | Шрифт | Вес | Межстрочный |
|---------|---------|--------|-------|-----|-------------|
| **H1 (Hero)** | 72–96px | 40–56px | Space Grotesk | 800 | 1.0–1.1 |
| **H2 (Секция)** | 48–64px | 32–40px | Space Grotesk | 800 | 1.1 |
| **H3 (Подзаголовок)** | 32–40px | 24–32px | Space Grotesk | 700 | 1.2 |
| **H4 (Карточка)** | 24–28px | 20–24px | Space Grotesk | 700 | 1.2 |
| **Body Large** | 18–20px | 16–18px | Inter | 400 | 1.6 |
| **Body** | 16px | 14–16px | Inter | 400 | 1.6 |
| **Caption** | 14px | 12–14px | Inter | 500 | 1.4 |
| **Button** | 16–18px | 14–16px | Space Grotesk | 700 | 1.0 |
| **Badge / Accent** | 14–18px | 12–16px | Unbounded | 600 | 1.2 |

### Подключение шрифтов

```html
<link href="https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;500;600;700;800&family=Inter:wght@300;400;500;600;700;800;900&family=Unbounded:wght@400;500;600;700;800;900&display=swap" rel="stylesheet">
```

### Правила типографики

- **letter-spacing:** -0.02em для заголовков (Space Grotesk), 0 для текста (Inter)
- **text-transform: uppercase** — только для кнопок и навигации
- **Кнопки** — Space Grotesk 700, uppercase, 16px padding: 16px 32px
- **Межстрочный** — 1.0–1.1 для заголовков, 1.5–1.6 для текста
- **Unbounded** — использовать точечно: числа в статистике, ключевые слова, логотип

## 3. Система теней (Neo-Brutalist)

### Спецификация теней

| Размер | CSS | Применение |
|--------|-----|-----------|
| **sm** | `2px 2px 0px 0px #000000` | Мелкие элементы: бейджи, теги, чекбоксы |
| **DEFAULT** | `4px 4px 0px 0px #000000` | Карточки, кнопки, поля ввода — основная тень |
| **lg** | `8px 8px 0px 0px #000000` | Hero-элементы, featured-карточки |
| **xl** | `12px 12px 0px 0px #000000` | Модальные окна, главный CTA |

### Правила теней

- Тени **ВСЕГДА** жёсткие (blur = 0). Никаких размытых теней (box-shadow с 3-м значением > 0)
- Цвет тени **ВСЕГДА** `#000000` — не серый, не прозрачный
- Направление: вправо-вниз (положительные значения x и y)
- При ховере: сдвиг элемента на величину тени + уменьшение тени → эффект «нажатия»

### CSS ховер-эффект (кнопка)

```css
.btn-neo {
  background-color: #CCFF00;
  color: #000000;
  border: 2px solid #000000;
  box-shadow: 4px 4px 0px 0px #000000;
  transition: all 0.15s cubic-bezier(0.4, 0, 0.2, 1);
}

.btn-neo:hover {
  transform: translate(2px, 2px);
  box-shadow: 2px 2px 0px 0px #000000;
}

.btn-neo:active {
  transform: translate(4px, 4px);
  box-shadow: none;
}
```

## 4. Border Radius

| Значение | px | Применение |
|----------|-----|-----------|
| **none** | 0px | Разделители, линии |
| **sm** | 4px | Кнопки, бейджи, теги, поля ввода |
| **DEFAULT** | 8px | Карточки, контейнеры, изображения |
| **md** | 12px | Модальные окна, большие карточки |
| **lg** | 16px | Максимум для neo-brutalist. Hero-блоки |
| **pill** | 9999px | ТОЛЬКО для округлых бейджей и индикаторов |

### Правило

Стиль Synapse — **жёсткий, графичный**. Углы 4-8px — стандарт. Радиус > 16px — исключение, а не норма. Никаких «облачных» элементов.

## 5. Рамки (Borders)

| Элемент | border |
|---------|--------|
| Кнопки | `2px solid #000000` |
| Карточки | `2px solid #000000` |
| Поля ввода | `2px solid #000000` |
| Разделители | `1px solid rgba(0,0,0,0.1)` |
| Акцентные рамки | `2px solid #CCFF00` или `2px solid #5D3FD3` |

Рамки — **обязательный** элемент neo-brutalist стиля. Без рамки элемент «парит». С рамкой — чёткий, графичный, весомый.

## 6. Логотип

### Текстовый логотип

```
SYN APSE
    ^^^^
    Lime highlight (#CCFF00)
```

- Написание: **SYN**APSE — первые 3 буквы обычным цветом (чёрный или белый), **APSE** — цветом `#CCFF00`
- Шрифт: **Unbounded**, вес 800
- Размер: адаптивный, но минимум 24px
- Отступ между буквами: -0.02em

### Варианты использования

| Контекст | Вид |
|----------|-----|
| Тёмный фон | `SYN` белый + `APSE` lime #CCFF00 |
| Светлый фон | `SYN` чёрный + `APSE` lime #CCFF00, с подложкой или рамкой |
| Монохром (печать) | Полностью чёрный, APSE — bold |
| Favicon | Буква **S** lime на чёрном фоне, или **SYN** — сокращение |

### Запреты по логотипу

- НЕ поворачивать
- НЕ менять шрифт логотипа
- НЕ выделять другие буквы кроме APSE
- НЕ использовать градиент в логотипе
- Минимальный отступ вокруг = высота буквы «S»

## 7. Tone of Voice

### Характер бренда

**Synapse — это крутой старший друг, который знает про AI всё и рассказывает просто, без снисхождения.**

| Мы | Мы НЕ |
|-----|--------|
| Прямые | Агрессивные |
| Уверенные | Высокомерные |
| Простые | Примитивные |
| Вдохновляющие | Восторженные |
| С юмором | Клоунские |
| Технически грамотные | Заумные |

### Тональность по каналам

| Канал | Тон |
|-------|-----|
| **Сайт** | Уверенный, лаконичный, заголовки-утверждения |
| **Соцсети** | Живой, с юмором, мемы ОК |
| **Родителям** | Спокойный, экспертный, конкретные факты |
| **Детям** | Равный, «на ты», без сюсюканья |
| **Документы (КП, PDF)** | Чистый, структурный, с цифрами |

### Формулы текста

**Заголовки — утверждения, не вопросы:**
- ДА: «AI — это не будущее. Это настоящее.»
- НЕТ: «Хотите ли вы, чтобы ваш ребёнок освоил AI?»

**Глаголы — активные:**
- ДА: «Создавай. Генерируй. Запускай.»
- НЕТ: «Вы получите возможность научиться...»

**Цифры — конкретные:**
- ДА: «9 инструментов. 4 проекта за месяц. 90 минут.»
- НЕТ: «Множество инструментов и проектов»

### Слова-маркеры бренда

**Используем:** AI-мышление, создавать, генерировать, промпт, нейросеть, проект, инструмент, скилл, запускать
**Избегаем:** программирование, кодинг, уроки (→ «занятия»), учить (→ «осваивать»), детишки, ребята, будущее (клише)

## 8. Иконография и графика

### Стиль иконок

- Линейные (stroke), толщина 2px
- Цвет: чёрный, lime или purple
- Размер: 24x24px (стандарт), 32x32px (крупные), 48x48px (hero)
- Библиотека: Lucide React или Heroicons (outline)

### Иллюстрации

- Стиль: плоские, графичные, с жёсткими тенями
- Цвета: только из палитры (lime, black, purple, white, grey)
- НЕ использовать: 3D-рендеры, стоковые фото с детьми, градиентные иллюстрации

### Фоновые паттерны

```css
/* Сетка */
background-image: url("data:image/svg+xml,%3csvg width='40' height='40' viewBox='0 0 40 40' xmlns='http://www.w3.org/2000/svg'%3e%3cpath d='M0 40L40 0H20L0 20M40 40V20L20 40' stroke='%23000000' stroke-opacity='0.05' stroke-width='1' fill='none'/%3e%3c/svg%3e");

/* Точки */
background-image: radial-gradient(circle, #000000 1px, transparent 1px);
background-size: 20px 20px;
```

## 9. Do's & Don'ts

### DO (делай так)

- Высокий контраст: чёрный + lime, белый + чёрный
- Жёсткие тени на ВСЕХ интерактивных элементах
- Крупная типографика в заголовках (72-96px hero)
- Много воздуха (padding 64-80px между секциями)
- Чёткая сетка (12 колонок, gap 24-32px)
- Uppercase для кнопок и навигации
- Рамки 2px solid #000 на карточках и кнопках
- Lime-подсветка ключевых слов в тексте

### DON'T (так нельзя)

- Размытые тени (box-shadow с blur > 0)
- Круглые углы > 16px (кроме pill-бейджей)
- Градиенты как основной приём (допустимы только как subtle-фон)
- Пастельные цвета — никогда
- Тонкие шрифты (< 400 weight) в заголовках
- «Детские» иллюстрации (мультяшные, kawaii, clip-art)
- Стоковые фото с улыбающимися детьми за компьютером
- Анимации дольше 0.3s для UI-элементов
- Больше 3 цветов на одном экране (lime + чёрный + 1 нейтральный)
