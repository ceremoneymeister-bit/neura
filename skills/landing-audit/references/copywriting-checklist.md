# Copywriting Checklist

## Scoring: 0-200 (10 categories × 20 pts)

### 1. Headline (20 pts)
- [ ] 6-12 слов (4 pts)
- [ ] Содержит конкретную выгоду, не абстрактное заявление (4 pts)
- [ ] Отвечает на "Что + Для кого + Почему сейчас" (4 pts)
- [ ] Не дублируется конкурентом (тест на своп) (4 pts)
- [ ] Message match: заголовок = обещание из рекламы/ссылки (4 pts)

### 2. Subheadline & Supporting Copy (20 pts)
- [ ] Подзаголовок добавляет НОВУЮ информацию (не эхо заголовка) (4 pts)
- [ ] Содержит конкретику: цифру, срок, квалификатор (4 pts)
- [ ] Средняя длина предложения ≤20 слов (4 pts)
- [ ] Уровень читаемости 5-7 класс (Flesch-Ru ≥60) (4 pts)
- [ ] Нет жаргона без объяснения (4 pts)

### 3. CTA Buttons (20 pts)
- [ ] 2-5 слов, ≤24 символа (3 pts)
- [ ] Начинается с глагола действия (3 pts)
- [ ] Первое лицо ("Получить мой..." а не "Ваш...") (+90% CTR) (3 pts)
- [ ] Описывает результат, не механику ("Записаться" а не "Отправить") (3 pts)
- [ ] Повторяется ≥3 раз на странице (hero + mid + pricing) (4 pts)
- [ ] Нет негативных слов рядом с CTA ("спам", "никогда") (4 pts)

### 4. Value Proposition (20 pts)
- [ ] Каждая фича имеет "чтобы..." (Feature → Benefit → Outcome) (5 pts)
- [ ] Все утверждения подкреплены цифрами (5 pts)
- [ ] Уникально: конкурент не может использовать тот же текст (5 pts)
- [ ] BAB структура: Before (боль) → After (желаемое) → Bridge (продукт) (5 pts)

### 5. Storytelling & Emotion (20 pts)
- [ ] Клиент = герой, бренд = наставник (4 pts)
- [ ] ≥3 из 7 эмоциональных триггеров: Trust, Curiosity, Desire, Fear, Guilt, Belonging, Value (4 pts)
- [ ] Язык из реальных отзывов/разговоров ЦА (4 pts)
- [ ] Эмоциональная арка: боль → эмпатия → решение (4 pts)
- [ ] ≥3 принципа Cialdini использованы (4 pts)

### 6. Social Proof (20 pts)
- [ ] ≥3 типа social proof (отзывы, лого, цифры, кейсы) (4 pts)
- [ ] Отзывы с конкретными результатами (4 pts)
- [ ] Отзывы с фото + имя + роль (3 pts)
- [ ] Совпадает с профилем ЦА (3 pts)
- [ ] Размещён ДО основного CTA (3 pts)
- [ ] Есть независимый источник (скриншот, платформа) (3 pts)

### 7. Objection Handling (20 pts)
- [ ] Топ 3-5 возражений ЦА адресованы (5 pts)
- [ ] Размещены ДО финального CTA (4 pts)
- [ ] 4 категории покрыты: "Что делает?", "Как работает?", "Почему доверять?", "Масштаб?" (4 pts)
- [ ] Гарантия заметна, на простом языке (4 pts)
- [ ] FAQ = реальные сомнения покупателя, не FAQ продукта (3 pts)

### 8. Readability (20 pts)
- [ ] Flesch-Ru ≥60 (5 pts)
- [ ] Gunning Fog-Ru <10 (5 pts)
- [ ] Подзаголовки каждые 3-4 абзаца (3 pts)
- [ ] Буллет-листы для перечислений (3 pts)
- [ ] Сканируемость: headline + subheads + CTA рассказывают историю (4 pts)

### 9. Information Architecture (20 pts)
- [ ] Порядок: Hero → Problem → Solution → Proof → Process → FAQ → CTA (5 pts)
- [ ] Above-fold: headline + value prop + CTA видны без скролла на mobile (5 pts)
- [ ] Trust signals распределены по странице (не в одном блоке) (5 pts)
- [ ] Плавные переходы между секциями (нет резких скачков тем) (5 pts)

### 10. Urgency & Conversion (20 pts)
- [ ] Urgency реальная (настоящие дедлайны, не фейк) (4 pts)
- [ ] Конкретные цифры ограничения (4 pts)
- [ ] "Почему сейчас" явно отвечено (4 pts)
- [ ] Одна конверсионная цель на странице (4 pts)
- [ ] Длина страницы соответствует сложности предложения (4 pts)

---

## Benchmarks
- Медианная конверсия лендинга: 6.6%
- Копия с языком клиента: +79.3% конверсия
- Отзывы над CTA: +64.5% конверсия
- Первое лицо в CTA: +90% CTR
- Удаление слова "спам": +19.5% подписок
