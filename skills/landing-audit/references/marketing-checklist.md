# Marketing & Psychology Checklist

## Scoring: 0-100 (14 dimensions, ~7 points each)

### 1. 5-Second Test (0-7)
- [ ] Через 5 секунд понятно: ЧТО продаётся, ДЛЯ КОГО, ПОЧЕМУ важно
- [ ] Заголовок самодостаточен (не требует подзаголовка для понимания)
- [ ] Визуал подкрепляет предложение (не stock photo)

### 2. Value Proposition Clarity (0-7)
- [ ] Уникальный механизм объяснён (ПОЧЕМУ работает, не просто "лучше")
- [ ] Нельзя поставить заголовок конкуренту без изменений (тест на своп)
- [ ] Benefit-to-Feature ratio ≥1:1 (выгод не меньше чем фич)

### 3. Social Proof / Consensus (0-7)
- [ ] ≥3 типа social proof (отзывы, логотипы, цифры, кейсы, пресса, рейтинги)
- [ ] Отзывы с конкретными результатами, фото, именем
- [ ] Social proof расположен ДО основного CTA

### 4. Authority (0-7)
- [ ] Экспертность подкреплена (сертификаты, опыт, медиа-упоминания)
- [ ] Данные из авторитетных источников (если есть статистика)

### 5. Scarcity & Urgency (0-7)
- [ ] Дедлайн/ограничение реальное (не фейковый таймер)
- [ ] Конкретные цифры ("3 места", дата, а не "ограничено")
- [ ] Причина "почему сейчас" озвучена

### 6. Reciprocity (0-7)
- [ ] Страница даёт ценность ДО запроса действия (инсайт, калькулятор, чеклист)

### 7. Commitment & Consistency (0-7)
- [ ] Есть микро-обязательства перед основным CTA (квиз, вопрос "да/нет")
- [ ] Градация: лёгкое действие → основное действие

### 8. Loss Aversion (0-7)
- [ ] Фрейминг потери ("что теряете без") присутствует
- [ ] Цена бездействия показана конкретно

### 9. Anchoring (0-7)
- [ ] Первая цифра на странице стратегически выбрана (старая цена, средний чек)
- [ ] Сравнительная рамка установлена до показа цены

### 10. Trust Architecture (0-7)
- [ ] Гарантия/возврат заметна и рядом с CTA
- [ ] Контактные данные видны (телефон, адрес, email)
- [ ] Реальные фото команды/автора (не стоковые)

### 11. Objection Handling (0-7)
- [ ] Топ-5 возражений ЦА адресованы на странице
- [ ] Возражения размещены ДО финального CTA
- [ ] FAQ отвечает на реальные вопросы покупателей, а не на вопросы про продукт

### 12. Emotional Arc (0-7)
- [ ] Страница следует PAS (Problem → Agitate → Solve)
- [ ] Эмоции перед логикой (сначала КАК изменится жизнь, потом фичи)
- [ ] Клиент = герой, бренд = наставник

### 13. CTA Strategy (0-7)
- [ ] Один основной CTA (нет конкурирующих кнопок)
- [ ] CTA повторяется ≥3 раз (hero + mid + pricing)
- [ ] Текст кнопки описывает результат ("Записаться за 9990₽" а не "Отправить")

### 14. Persona Fit (0-7)
- [ ] Страница говорит к ОДНОЙ конкретной аудитории
- [ ] Язык копии = язык ЦА (не маркетинговый жаргон)
- [ ] Отзывы от людей, похожих на целевого клиента
