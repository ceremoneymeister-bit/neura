# landing-page

Unified Landing Page Creation Workflow — оркестратор для создания конверсионных лендингов.

## Что делает

Объединяет 10 скиллов экосистемы в последовательный процесс из 6 фаз:

1. **Discovery & Brief** — сбор контекста, определение awareness-уровня
2. **Strategy & Psychology** — выбор ментальных моделей и дизайн-направления
3. **Design System** — палитра, типографика, токены (DFII ≥ 8)
4. **Content Creation** — копирайтинг, SEO, русская типографика
5. **Implementation** — React + Vite + Tailwind v4, mobile-first
6. **Verification** — LPQI-скоринг (≥40/55), build, pre-delivery checklist

## Когда использовать

- Создание лендинга с нуля
- Редизайн существующей конверсионной страницы
- Промо-страница для продукта, события, сервиса
- Любой запрос вида "сделай лендинг", "нужна посадочная страница"

## Ключевые особенности

- **Hard Gates** — Brief Lock и DFII ≥ 8 предотвращают работу без основы
- **LPQI** — 5-осевой скоринг качества (Design, Copy, SEO, Psychology, Technical)
- **Anti-Patterns** — 7 проверок перед сдачей
- **Не дублирует** — вызывает существующие скиллы в правильном порядке
