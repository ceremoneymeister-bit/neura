# Сценарий 3: Голос → действие

## Контекст
Самый быстрый сценарий (30 сек). Демонстрирует что агент ДЕЙСТВУЕТ, а не советует. Голосовое сообщение → реальное действие (событие в календаре, сообщение человеку).

## Инструменты
- `scripts/icloud_calendar.py` — добавление событий в календарь
- `scripts/tg-send.py` — отправка сообщений через Telegram userbot

## Алгоритм выполнения

### Шаг 1: Приём голосового (автоматически)
Голосовое транскрибируется через Whisper (voice handler бота). Текст поступает агенту.

### Шаг 2: Извлечение интента
Из транскрипции определить:
- **Тип действия:** событие / сообщение / напоминание / задача
- **Параметры:** кому, когда, что

### Шаг 3: Выполнение действия

**Если событие в календаре:**
```bash
python3 /root/Antigravity/scripts/icloud_calendar.py add \
  --title "[Название]" \
  --date "YYYY-MM-DD" \
  --time "HH:MM" \
  --calendar "Работа"
```

**Если сообщение человеку:**
```bash
python3 /root/Antigravity/scripts/tg-send.py [user] "[Текст сообщения]"
```

**Если и то, и другое — выполнить оба действия.**

## Примеры голосовых для демо

**Простой:**
> "Запиши встречу с Иваном на завтра в 15:00"

**Составной:**
> "Напомни мне позвонить в банк завтра утром и запиши это в календарь"

**Бизнес:**
> "Напиши Алексею что встреча переносится на четверг и добавь в календарь"
