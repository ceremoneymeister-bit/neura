# Сценарий 2: Контент-машина

## Контекст
Быстрый и впечатляющий сценарий. За 60 секунд — 5 готовых постов для бизнеса собеседника. Можно публиковать сразу.

## Зависимые скиллы (прочитать перед выполнением)
1. `social-content` — стратегия контента, хуки, форматы
2. `copywriting` — убедительные тексты
3. `marketing-psychology` — психологические триггеры

## Алгоритм выполнения

### Шаг 1: Контекст (10 сек)
Из запроса извлечь:
- Ниша / бизнес
- Целевая аудитория (если не указана — определить по нише)
- Платформа (по умолчанию: Telegram + Instagram)

### Шаг 2: Генерация 5 постов (40 сек)

Каждый пост — другой формат:

| # | Формат | Психология | Длина |
|---|--------|------------|-------|
| 1 | 🔥 Провокация | Разрыв шаблона | 3-5 предложений |
| 2 | 📖 История | Нарратив + мораль | 5-7 предложений |
| 3 | 📋 Список | Полезность + сохранение | 7-10 пунктов |
| 4 | 💡 Инсайт | Экспертность + авторитет | 3-5 предложений |
| 5 | ❓ Вовлечение | Вопрос + дискуссия | 3-4 предложения + вопрос |

### Требования к каждому посту:
- **Хук в первой строке** — цепляет за 1.5 сек (по формулам из social-content)
- **Конкретика** — цифры, примеры, имена (не абстракции)
- **CTA** — каждый пост заканчивается действием
- **Без воды** — каждое предложение несёт смысл
- **Эмодзи** — умеренно, по 2-3 на пост
- **Адаптация под нишу** — терминология и боли целевой аудитории

## Советы для wow-эффекта
- Посты должны быть РЕАЛЬНО хорошими — не шаблонные, а цепляющие
- Использовать конкретику ниши собеседника (не generic "ваш бизнес")
- Если собеседник называет конкретный продукт — использовать его в постах
- Хуки — самая важная часть. Потратить на них 50% усилий
