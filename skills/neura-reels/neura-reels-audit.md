## 2. Чего не хватает для российского рынка

### 2.1 VK Клипы — отсутствуют в platform-rules.md
**Проблема:** VK — основная площадка Максима (2200 подп), но VK Клипы не описаны.
**Рекомендация:** Добавить секцию VK Clips в `platform-rules.md`:

```
## VK Клипы
| Параметр | Значение |
|----------|----------|
| Оптимальная длительность | 15-60 сек |
| Формат | 9:16 вертикальный |
| Описание | До 2048 знаков. Хештеги в тексте |
| Музыка | VK Music library (не Instagram trending!) |
| Алгоритм | Приоритет: досмотр + взаимодействие |
| Аудитория | Старше, чем Instagram. 25-45 лет, предприниматели |
| Хештеги | #бизнес #ии #автоматизация #нейра |
| Особенность | Можно добавлять ссылку в описании (Instagram нельзя!) |
```

### 2.2 Яндекс.Дзен — не упомянут
**Проблема:** Дзен — растущая платформа в РФ, алгоритмическая раздача.
**Рекомендация:** Добавить как опциональную платформу. Короткие видео в Дзене набирают обороты.

### 2.3 Специфика российской аудитории
**Проблема:** Hooks и примеры универсальные, не учитывают боли РФ-предпринимателей.
**Рекомендации:**
- Добавить секцию "Болевые точки РФ-аудитории" в product-dna.md:
  - VPN и блокировки AI-сервисов — ГЛАВНАЯ боль
  - Санкции: многие SaaS недоступны, нужны альтернативы
  - Скепсис к инфобизу: "ещё один курс / ещё один бот"
  - Telegram = норма жизни (не альтернатива, а основной мессенджер)
  - Цены в рублях, сравнение с зарплатами (маркетолог 60-120к/мес)

### 2.4 Отсутствие адаптаций по платформам
**Проблема:** Секция "Адаптация одного сценария под 3 платформы" есть, но нет конкретных шаблонов адаптации.
**Рекомендация:** Для каждого сценария генерировать 3 версии:
- Instagram Reels: полная версия (45-60 сек)
- VK Клипы: полная версия + ссылка в описании (!)
- YouTube Shorts: ≤ 58 сек + SEO-описание
- TikTok: сокращённая (30-45 сек)

## 5. Форматы рилсов для РФ

### YouTube Shorts — алгоритм другой
YouTube Shorts продвигает через:
- Retention (досмотр) — приоритет №1
- CTR обложки — текст должен быть читаемый
- SEO-описание — YouTube индексирует текст (в отличие от Instagram)

**Рекомендация для Shorts:** Добавлять полноценное SEO-описание (100-200 слов) с ключевыми словами: "AI для бизнеса", "AI-агент в Telegram", "автоматизация бизнеса".

### VK Клипы — аудитория другая
VK аудитория:
- Старше (25-45 vs 18-30 в Instagram)
- Более скептичная к "хайпу"
- Ценит конкретику и пользу > wow-эффект
- Привыкла к длинным текстам (VK посты = длинные)

**Рекомендация:** В VK Клипах делать акцент на:
- Практическую пользу (не "вау", а "полезно")
- Конкретные цифры и кейсы
- Ссылку в описании (CTA = перейди, а не "напиши слово")
