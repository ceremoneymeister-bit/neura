# neura-reels

Генерация готовых к съёмке сценариев Reels/Shorts для продажи AI-агентного сервиса Нейра. Каждый скрипт включает: хук с психологическим триггером, потактовую раскладку, визуальные указания, текст на экране, голосовой текст и CTA с механикой конверсии.

## Files

| File | Purpose |
|------|---------|
| `SKILL.md` | Основной workflow: бриф → генерация → скоринг → выдача |
| `references/hooks-catalog.md` | 24 формулы хуков в 6 категориях |
| `references/script-templates.md` | 6 форматов сценариев с таймингами |
| `references/product-dna.md` | Факты о Нейре для скриптов |
| `references/brand-voice.md` | Голос бренда Максима Белоусова |
| `references/filming-guide.md` | Как снимать каждый формат |

## Triggers

This skill activates when:
- User asks to create a Reel script for Neura
- User says "рилс", "reels", "сценарий", "сними видео"
- User needs video content to sell AI agent services
- User wants a batch of scripts for content plan

## Quick Start

Просто скажи: "Сделай рилс для Нейры" — скилл сгенерирует 3 сценария в разных форматах с полной раскладкой по секундам, визуалом и CTA.

Для конкретного запроса: "Рилс про то, как Нейра делает сайты за 2 минуты" — один скрипт в формате Speed Demo.
