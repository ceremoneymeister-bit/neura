# Brand Voice Максима Белоусова — правила голоса

> Каждый скрипт должен звучать как Максим.
> Не как корпорация, не как блогер, не как инфобизнесмен — как Максим.

---

## Ядро голоса

**Три слова:** Прямой. Доказательный. Энергичный.

Максим — практик, который 3 года работает с нейросетями, прошёл путь от маркетплейсов до AI, и говорит о технологиях через результаты, а не через теорию.

---

## Паттерны, которые ВСЕГДА работают

### 1. Proof-First (сначала доказательство)
Максим никогда не начинает с обещаний. Он начинает с демонстрации.
- ✅ "Смотри: я отправил голосовое — через 2 минуты получил сайт"
- ❌ "Представьте, как было бы здорово иметь AI-помощника"

### 2. Конкретные числа
Каждое утверждение подкреплено метрикой.
- ✅ "5 постов за 60 секунд"
- ❌ "Быстро генерирует контент"

### 3. ChatGPT как точка отсчёта
Аудитория знает ChatGPT. Максим всегда отталкивается от него.
- ✅ "ChatGPT забывает. Нейра помнит"
- ❌ "Наш AI лучше других решений на рынке"

### 4. «Пока ты спишь»
Ключевая метафора автономности.
- ✅ "6 AI-директоров работают пока я сплю"
- ❌ "Система работает автономно в фоновом режиме"

### 5. Действие > Слова
Глаголы действия, не состояния.
- ✅ "Делает сайты. Пишет посты. Анализирует конкурентов."
- ❌ "Способен создавать различный контент и проводить аналитику"

### 6. Низкий порог входа
CTA всегда лёгкие, без обязательств.
- ✅ "Напиши 'НЕЙРА' — скину демо"
- ❌ "Оставьте заявку на нашем сайте для получения коммерческого предложения"

## Тон и ритм

### Тон
- **Уверенный** — говорит как человек, который уже видел результат
- **Дружеский** — на "ты", без дистанции
- **Энергичный** — не кричит, но говорит с внутренним огнём
- **Прямой** — без вводных конструкций, без реверансов

### Ритм речи для видео
- Предложения: 5-9 слов максимум
- Паузы: после каждого сильного утверждения (0.5-1 сек)
- Темп: средний-быстрый, ускоряется на перечислениях
- Энергия: нарастает к финалу, пик — на главном инсайте
- Голос: уверенный средний регистр, без "вещания" (не диктор новостей)

### Обращение к зрителю
- "Ты" — всегда (не "вы", не "коллеги", не "друзья")
- "Смотри" — когда нужно привлечь внимание к демо
- "Знаешь, что?" — перед контринтуитивным фактом
- Без "Привет", "Всем привет", "Добро пожаловать" — начинаем с хука

## Словарь: используй / не используй

### ✅ Используй
| Слово/фраза | Контекст |
|-------------|----------|
| Нейра | Название продукта |
| бот / AI-бот | Простое описание |
| AI-отдел | Метафора масштаба |
| навыки (не скиллы) | Для аудитории — "скиллы" непонятно |
| помнит | Ключевое отличие от ChatGPT |
| делает (не может) | Proof-first: он УЖЕ делает |
| Telegram | Точка входа |
| за 2 минуты | Скорость |
| 24/7 | Доступность |
| пока ты спишь | Автономность |
| попробуй / напиши | CTA |

### ❌ Не используй
| Слово/фраза | Почему |
|-------------|--------|
| нейросеть | Устаревшее для 2026 |
| искусственный интеллект (полностью) | Слишком формально, "AI" — нормально |
| платформа | Корпоративный жаргон |
| решение / продукт | Корпоративный жаргон |
| уникальный / инновационный | Пустые прилагательные |
| лучший на рынке | Необоснованное утверждение |
| монетизация / масштабирование | Инфобиз-жаргон |
| воронка / конверсия | В рилсе — для зрителя, не для маркетолога |
| промпт / prompt | Аудитория не понимает |
| Claude / GPT-4 / API | Технический жаргон |

## Эмоциональный диапазон

| Эмоция | Когда использовать | Как звучит |
|--------|-------------------|------------|
| Удивление | Demo-хуки: "Смотри что может" | Широкие глаза, ускорение темпа |
| Уверенность | Myth-хуки: "Это миф. Нет." | Ровный тон, пауза после |
| Эмпатия | Story-хуки: "Он терял клиентов" | Мягче, медленнее, наклон к камере |
| Вызов | Challenge-хуки: "А ты до сих пор..." | Прямой взгляд, чуть громче |
| Спокойная сила | CTA: "Попробуй. Что терять?" | Ровно, без давления |

## Что Максим НИКОГДА не скажет
- "Я гуру / эксперт №1" — позиционирование через результаты, не титулы
- "Поверьте мне" — доверие через демо, не через слова
- "Это изменит вашу жизнь" — инфобиз-клише
- "Только сегодня / осталось 3 места" — манипулятивный дефицит
- Негатив о конкретных людях/компаниях (кроме общего сравнения с ChatGPT)
