## Format 1: SPEED DEMO (самый сильный для конверсии)

**Суть:** Показать, как Нейра делает что-то впечатляющее в реальном времени.
**Длительность:** 45-60 секунд
**Стиль съёмки:** Скринкаст + face cam bubble в углу
**Психология:** Proof > Promise. Зритель видит своими глазами.

```
[0-3 сек] ХУК
📹 Face cam, уверенный взгляд
🎙️ "{Хук из категории 1 — Demo}"
📝 Текст хука крупно на экране

[3-6 сек] КОНТЕКСТ
📹 Переход на скринкаст Telegram
🎙️ "Открываю бота. Отправляю одно сообщение."
📝 —

[6-35 сек] ДЕМО
📹 Скринкаст: набор сообщения → отправка → ожидание → результат появляется
🎙️ Комментарий в реальном времени: "Набираю... отправляю... смотрите..."
📝 Ключевые моменты подсвечивать зумом + текстом
⚡ Ускорить ожидание в 2-4x, оставить реальную скорость на wow-моментах
🔔 Звук "ding" когда результат появляется

[35-42 сек] РЕЗУЛЬТАТ
📹 Показать готовый результат крупно (сайт/посты/анализ)
🎙️ "[Время]. Без [дизайнера/копирайтера/аналитика]. Через Telegram."
📝 "{Время} → {Результат}" крупно

[42-50 сек] УСИЛЕНИЕ
📹 Face cam
🎙️ "И это один навык из 113. Он работает 24/7."
📝 "113 навыков • 24/7 • Telegram"

[50-60 сек] CTA
📹 Face cam + текст на экране
🎙️ "Хочешь попробовать на своём бизнесе? Напиши '{СЛОВО}' в комментариях"
📝 "Напиши '{СЛОВО}' ↓"
```

**Loop point:** Последний кадр (CTA текст на экране) → визуально похож на первый кадр (лицо + текст).

## Format 7: META-AI (AI сам создаёт контент)

**Суть:** Мета-рилс — AI-агент сам создал этот рилс, сам написал пост, сам запостил. Сам факт создания контента = контент. Это доказательство "на стероидах": зритель смотрит продукт, который создан продуктом.
**Длительность:** 45-60 секунд
**Стиль съёмки:** Скринкаст Telegram → готовый результат (пост/видео/сторис)
**Психология:** Meta-Proof + Inception Effect (продукт рекламирует себя сам) + Wow Factor

**Почему работает:**
Зритель видит замкнутый цикл: AI-агент → создал сценарий → создал пост → запостил → ты сейчас смотришь результат. Это самый сильный proof of concept — не "может сделать", а "уже сделал, и ты смотришь это".

```
[0-3 сек] ХУК
📹 Экран телефона: Instagram / Telegram
🎙️ "Этот рилс создал не я. Его создал мой AI-агент."
📝 "Этот рилс создал AI" — крупно

[3-8 сек] КОНТЕКСТ
📹 Скринкаст: открытие Telegram → сообщение боту
🎙️ "Я написал одно сообщение: 'Сделай рилс про [тема]'."
📝 Показать сообщение крупно

[8-20 сек] ПРОЦЕСС
📹 Скринкаст: бот генерирует сценарий → показать текст → бот создаёт подпись для поста
🎙️ "Сценарий. Подпись. Хештеги. Время публикации. Всё — за 2 минуты."
📝 Подсвечивать каждый элемент зумом
⚡ Ускорение 2-3x на ожидании

[20-30 сек] РЕЗУЛЬТАТ
📹 Показать готовый пост / рилс / сторис
🎙️ "Вот готовый пост. Написан, оформлен, запланирован. Я не трогал клавиатуру."
📝 "0 минут ручной работы"

[30-42 сек] МАСШТАБ
📹 Face cam
🎙️ "И так — каждый день. Контент-план на неделю. Публикация по расписанию. Аналитика. Всё через одного бота в Telegram."
📝 "7 дней • авто-контент • 1 бот"

[42-50 сек] ПЕРЕВОРОТ (Peak)
📹 Face cam, пауза, прямой взгляд
🎙️ "То, что ты сейчас смотришь — тоже создал он."
📝 "Inception: AI создал рилс про AI"

[50-60 сек] CTA
📹 Face cam + текст
🎙️ "Хочешь так же? Напиши '{СЛОВО}' — покажу как работает."
📝 "Напиши '{СЛОВО}' ↓"
```

**Loop point:** "Этот рилс создал AI" (начало) → "То, что ты смотришь — тоже создал он" (конец) — зритель пересматривает.

**Варианты META:**
- "AI ведёт мой Instagram 30 дней. Вот результат" — с цифрами роста
- "Я не писал ни одного поста за неделю. Мой бот — писал" — показать ленту
- "AI создал этот пост, написал к нему текст и сам опубликовал" — полный цикл

## Format 8: NICHE LIFEHACK (узкоспециализированный viral)

**Суть:** Конкретный лайфхак для узкой проблемы, которую знает МНОГО людей, но решение знают ЕДИНИЦЫ. Не продаёт напрямую — привлекает МАССОВУЮ аудиторию, часть которой конвертируется в подписчиков и клиентов.
**Длительность:** 30-60 секунд
**Стиль съёмки:** 100% скринкаст с голосом за кадром
**Психология:** Practical Value (Berger) + Specificity = Shareability + Search Intent

**Почему это САМЫЙ вирусный формат:**
Кейс: рилс "Как получить доступ к нейросетям в России через App Store" набрал 200k+ просмотров и привёл 1000+ подписчиков. Причина: узкая проблема (Apple ID + подписки + регион) × огромная аудитория (все пользователи iPhone в России) × пошаговое решение = максимальный save + share.

**Принцип viral niche:**
```
VIRAL = (Размер аудитории с проблемой) × (Конкретность решения) × (Простота повторения)
```

Если хотя бы один множитель = 0, рилс не залетит. Все три должны быть высокими.

**Как находить темы:**
1. **Свои боли** — что ты сам гуглил в последнюю неделю и не мог найти нормальный ответ?
2. **Комментарии** — какие вопросы задают в комментариях под чужими рилсами в нише?
3. **Поиск** — что люди ищут в Google/YouTube по теме AI/нейросети/автоматизация?
4. **Ограничения** — какие ограничения/блокировки/проблемы есть у пользователей в России (VPN, блокировки, санкции, App Store)?
5. **"А я не знал"** — что знают технари, но не знают обычные пользователи?

```
[0-3 сек] ХУК-ПРОБЛЕМА
📹 Скринкаст: экран с проблемой (ошибка, блокировка, запрет)
🎙️ "{Конкретная проблема одним предложением}"
📝 Текст проблемы крупно

[3-8 сек] КОНТЕКСТ
📹 Скринкаст: показать почему стандартное решение НЕ работает
🎙️ "Обычно говорят: [стандартный совет]. Но если у тебя [конкретное ограничение] — это не работает."
📝 "❌ Стандартный способ не работает"

[8-35 сек] ПОШАГОВОЕ РЕШЕНИЕ
📹 Скринкаст: каждый шаг крупно, медленно, понятно
🎙️ "Шаг 1: [действие]. Шаг 2: [действие]. Шаг 3: [действие]."
📝 Нумерация шагов на экране: "Шаг 1/4", "Шаг 2/4"...
⚡ НЕ ускорять! Зритель должен успеть повторить или записать
🔑 Каждый шаг = отдельный кадр с чётким переходом

[35-45 сек] РЕЗУЛЬТАТ
📹 Скринкаст: готовый результат (работает!)
🎙️ "Готово. Теперь [что стало возможно]."
📝 "✅ Работает!"

[45-55 сек] БОНУС-МОСТ (опционально, для перевода в Нейру)
📹 Face cam или скринкаст
🎙️ "Кстати, если хочешь, чтобы AI делал такие вещи за тебя — у меня есть бот."
📝 — (не нужно, мягкая подводка)

[55-60 сек] CTA
📹 Face cam или текст на экране
🎙️ "Сохрани, чтобы не потерять. И подпишись — таких лайфхаков будет много."
📝 "💾 Сохрани • Подпишись"
```

**CTA для niche lifehack — ДРУГОЙ, чем у продающих рилсов:**
- НЕ "напиши слово" — здесь цель = охват и подписчики, не лиды
- "Сохрани" — главное действие (saves = algorithmic boost)
- "Подпишись" — вторичное
- Мост к продукту — мягкий, в конце, без давления

**Категории niche-тем для Максима / AI-ниши:**
| Категория | Примеры тем |
|-----------|-------------|
| Блокировки РФ | Доступ к ChatGPT/Claude из России, App Store регион, VPN для AI |
| AI-лайфхаки | Скрытые функции ChatGPT, промпты-однострочники, бесплатные AI-инструменты |
| Telegram-хаки | Скрытые настройки, боты, автоматизации, которые мало кто знает |
| Бизнес-инструменты | Бесплатные замены дорогому софту через AI, автоматизации без кода |
| Apple/Android | Скрытые функции телефона + AI, Siri Shortcuts + AI, виджеты |

**Правила niche lifehack:**
1. ОДНА проблема = ОДНО решение. Не "5 способов", а "вот как"
2. Скринкаст > face cam. Зритель хочет видеть КАК, а не КОГО
3. Медленно! Зритель повторяет шаги. Быстрый скринкаст = отписка
4. Субтитры ОБЯЗАТЕЛЬНЫ — 85% смотрят без звука, а здесь важна каждая деталь
5. Заголовок в начале = поисковый запрос ("Как получить ChatGPT в России")
6. Бонус-мост к продукту — ТОЛЬКО в конце, мягко, не обязательно
7. Количество шагов ≤ 5 — больше = потеря внимания

**Длительность:** 45-60 секунд
**Стиль съёмки:** Скринкаст-компиляция + voice over
**Психология:** Variable Reward (каждая задача = новый wow) + Completeness Bias (хочется досмотреть все)

```
[0-3 сек] ХУК

🎙️ "5 задач, которые мой AI решил сегодня"
📝 "5 задач AI за 1 день"

🎙️ "Первая: [описание]. [Время]."
📝 "1️⃣ [Задача] → [Время]"
⚡ Ускорение 3-4x

[12-21 сек] ЗАДАЧА 2
📹 Скринкаст: другая задача
🎙️ "Вторая: [описание]. [Время]."
📝 "2️⃣ [Задача] → [Время]"

[21-30 сек] ЗАДАЧА 3
📹 Скринкаст: третья задача
🎙️ "Третья: [описание]."

[30-39 сек] ЗАДАЧА 4 (скринкаст)
📹 Скринкаст: четвёртая
🎙️ "И ещё: [описание]."
```

## Правила адаптации шаблонов

1. **Длительность гибкая** — можно сжать 60-секундный шаблон до 45 (убрать УСИЛЕНИЕ) или растянуть до 90 (добавить ещё один пример)
2. **Хук взаимозаменяем** — любой хук из каталога подходит к любому формату. Подбирай по теме
3. **Пропорции железные** — Hook: 5-7% | Content: 60-70% | Peak: 10-15% | CTA: 15-20%
4. **CTA-слово** — менять каждые 5-10 рилсов. Примеры: НЕЙРА, БОТ, АГЕНТ, ИИ, АВТОМАТИЗАЦИЯ, САЙТ
5. **Batch production** — снимать 3-5 рилсов за одну съёмку. Менять хук и формат, сохранять энергию
