# auto-funnel

## Purpose

Проектирование автоворонок продаж: цепочка продуктов и коммуникаций, работающая на автопилоте. Автоворонка конвертирует не только горячих (2-3%), но и теплых и холодных клиентов, увеличивая выручку при том же трафике.

Этот скилл нужен чтобы:
- Спроектировать полную воронку от трафика до повторных продаж
- Выбрать подходящий шаблон воронки под конкретный бизнес
- Прописать каждый этап: страницы, письма, OTO, допродажи
- Задать метрики и benchmarks для оптимизации

---

## Концепция

**Автоворонка = "соковыжималка для трафика"** (метафора НЕЧ20).

Без воронки: обрабатываете только 2-3% горячих клиентов, 97% теряете.
С воронкой: конвертируете и теплых и холодных через промежуточные шаги.

Воронка продаж -- это последовательность продуктов, предложений и инструментов маркетинга, объединенных в единую модель, проходя через которую клиент совершает покупки.

**"Нефтяная вышка"** -- метафора НЕЧ20: строишь один раз, качаешь постоянно. Автоворонка окупается через 2-3 месяца и работает годами.

Успех зависит от:
- Трафика (теплота, количество, цена)
- Продуктовой линейки (ядро воронки)
- Упаковки продуктов (понятность ценности)
- Коммуникаций с клиентом (email, чат-боты, звонки)

## 6 шаблонов воронок

### 1. Квалифицирующая
```
[Трафик] -> [Квиз/Опрос] -> [Персонализированное предложение] -> [Звонок менеджера] -> [Продажа]
```
**Когда:** услуги с индивидуальным подходом, B2B, высокий чек.
**Ключевое:** квиз сегментирует аудиторию, предложение адаптируется под ответы.

### 2. Платный вход (Tripwire)
```
[Трафик] -> [Лид-магнит] -> [Трипвайр 490-990р] -> [Основной продукт] -> [Максимизатор/VIP]
```
**Когда:** инфопродукты, курсы, SaaS. Трипвайр окупает трафик.
**Ключевое:** низкий порог входа, человек становится покупателем.

### 3. Рекурринг (подписка)
```
[Трафик] -> [Бесплатный контент/триал] -> [Подписка] -> [Автопродление] -> [Апгрейд тарифа]
```
**Когда:** SaaS, клубы, регулярные услуги, медиа.
**Ключевое:** LTV растет с каждым месяцем.

### 4. Полная (классическая)
```
[Трафик] -> [Лид-магнит] -> [Email-серия 5-7 писем] -> [Видеокурс/Вебинар] -> [Оффер] -> [OTO] -> [Допродажа]
```
**Когда:** сложные продукты, длинный цикл решения, высокий чек.
**Ключевое:** максимальный прогрев, высокая конверсия. Пример WillGood: 92.6% прохождение воронки.

### 5. Вебинарная
```
[Трафик] -> [Регистрация на вебинар] -> [OTO на спасибо-странице] -> [Вебинар] -> [Оффер] -> [Дожим (email + звонки)]
```
**Когда:** экспертные продукты, обучение, консалтинг.
**Ключевое:** вебинар = демонстрация экспертизы + оффер в конце.

### 6. Каскадная (downsell)
```
[Трафик] -> [Мини-курс бесплатно] -> [Основной оффер] -> [Даунселл (лайт-версия)] -> [Звонок менеджера]
```
**Когда:** когда важно не потерять тех, кто не готов к основному продукту.
**Ключевое:** fallback-ветка для каждого "нет".

## OTO (One-Time Offer)

### Принцип
Пик вероятности покупки = момент нажатия кнопки. Каждая секунда после -- вероятность падает. Страница "Спасибо" = упущенная возможность. Вместо нее -- OTO.

### 6 атрибутов идеального OTO
1. **Момент** -- показывается сразу после целевого действия (регистрации, заявки, покупки)
2. **Связь с предыдущим шагом** -- предложение логически продолжает то, что человек только что сделал
3. **Дизайн-консистентность** -- та же стилистика, что и предыдущая страница (не выглядит как сторонний сайт)
4. **Супер-ценность** -- цена ощущается как "воровство" (скидка 60-90% от обычной)
5. **One-time + таймер** -- предложение исчезает навсегда, создает urgency
6. **Кристально ясный CTA** -- одна кнопка, одно действие, без путаницы

### 6 тактик OTO
| Тактика | Описание | Пример |
|---------|----------|--------|
| **Трипвайр** | Дешевый продукт (67-990р) для превращения лида в покупателя | Записи мастер-класса за 490р |
| **Апселл** | Улучшенная версия того, что купили | PRO-версия, VIP-доступ |
| **Кросс-селл** | Дополнительный продукт из смежной категории | К курсу по шитью -- набор выкроек |
| **Бандл** | Пакет из нескольких продуктов со скидкой | 3 курса по цене 1.5 |
| **Флеш-распродажа** | Резкая скидка на ограниченное время | -70% только 15 минут |
| **Early bird** | Скидка за раннее решение | Первые 50 мест по 5990р вместо 9990р |

### OTO может быть каскадным
OTO-1 -> OTO-2 -> OTO-3 (бесконечное количество). При отказе от OTO -- предложить более легкий шаг (квиз, подписка на соцсети, даунселл).

## Email-цепочка (прогревающая серия)

| # | День | Тема | Содержание |
|---|------|------|-----------|
| 1 | 0 | Благодарность + доставка | Спасибо за [действие]. Вот ваш [лид-магнит]. Что вас ждет дальше. |
| 2 | 1 | История + ценность | Личная история: как [проблема] привела к [решению]. Инсайт для читателя. |
| 3 | 2 | Кейс клиента | Конкретный результат: [имя] получил [результат] за [срок]. До/после. |
| 4 | 3 | Возражение + ответ | Самое частое возражение. Честный разбор. Доказательства. |
| 5 | 4-5 | Оффер + дедлайн | Предложение. Что входит. Бонусы. Дедлайн (конкретная дата). |
| 6 | 5-7 | Последний шанс | Напоминание. Последний день. Что теряет, если не купит. |

**Правила:**
- Каждое письмо -- одна мысль, один CTA
- Тема письма = главная интрига (open rate зависит на 80% от темы)
- Персонализация: имя, сегмент, предыдущие действия
- P.S. -- второй по читаемости элемент после заголовка

## Workflow (5 фаз)

### Phase 1: Анализ (ОБЯЗАТЕЛЬНО первой)
- Определить основной продукт и его цену
- Определить ЦА: кто, какая боль, какой результат хотят
- Определить уровни доверия аудитории (холодные/теплые/горячие)
- Сформировать продуктовую линейку (product ladder):
  - Лид-магнит (бесплатно)
  - Трипвайр (490-1990р)
  - Основной продукт
  - Максимизатор / VIP
  - Рекурринг (если применимо)

### Phase 2: Выбор шаблона
- Проанализировать бизнес-модель клиента
- Выбрать 1 из 6 шаблонов (см. выше)
- Адаптировать под специфику ниши
- Определить fallback-ветки (что если "нет" на каждом этапе)

### Phase 3: Проектирование этапов
Для каждого этапа воронки определить:
- Тип страницы/коммуникации
- Контент (что видит/читает пользователь)
- CTA (одно целевое действие)
- OTO (если применимо)
- Метрика успеха (конверсия этапа)

### Phase 4: Тексты и контент
- Лид-магнит: заголовок + обещание результата
- Лендинги: -> скилл `copywriting`, `headline-lab`
- Email-цепочка: 6 писем по шаблону выше
- OTO: видео-сценарий или текст страницы
- Вебинар: -> скилл `webinar-script` (если есть)

### Phase 5: Метрики и оптимизация
Установить benchmarks и точки контроля:

| Метрика | Benchmark | Критичная зона |
|---------|-----------|----------------|
| Конверсия лендинга (лид-магнит) | 30-50% | < 20% |
| Open rate email | 20-40% | < 15% |
| Click rate email | 3-7% | < 2% |
| Посещаемость вебинара (от рег.) | 30-50% | < 20% |
| Конверсия в продажу | 5-15% | < 3% |
| Прохождение воронки | 80-95% | < 60% |
| OTO конверсия | 5-15% | < 3% |

## Anti-patterns

| Ошибка | Почему плохо | Что делать |
|--------|-------------|-----------|
| Воронка без лид-магнита (сразу продажа) | Конверсия 1-2%, теряете 98% трафика | Добавить бесплатный промежуточный шаг |
| OTO без таймера | Нет срочности, откладывают и забывают | Таймер 15-30 мин, "предложение исчезнет навсегда" |
| Страница "Спасибо" без следующего шага | Упущенная возможность в пике вовлеченности | Заменить на OTO или следующее действие |
| Email-цепочка без персонализации | Массовость = игнор | Имя, сегмент, контекст предыдущего действия |
| Нет fallback-ветки (что если не купил?) | Потерянные лиды навсегда | Альтернативное предложение, ретаргетинг, даунселл |
| Слишком много CTA на одной странице | Паралич выбора, падение конверсии | Одна страница = один CTA |
| Воронка без метрик | Нельзя оптимизировать то, что не измеряешь | Установить benchmarks ДО запуска |
| Копирование чужой воронки без адаптации | Разные ниши = разное поведение ЦА | Шаблон + адаптация под свою аудиторию |

## Кейсы из источников

### WillGood (франшиза автосервисов)
- Шаблон: Полная воронка
- Путь: email -> лендинг -> видеокурс -> заявка -> менеджер -> продажа
- Fallback: чеклист -> прогревающие письма -> видеокурс
- Результат: 1674 регистрации -> 1551 прошли (92.6%) -> 32 заявки -> 2 продажи -> 2.5М руб.

### Бизнес Молодость (OTO за 67 руб)
- Шаблон: Вебинарная + OTO
- Путь: лендинг -> подписка -> OTO (67 руб) -> видеокурс
- Результат за 24 дня: 54 058 человек в воронке, 50% конверсия в заявку, 17 386 оплат, 17.5М руб.

### Школа шитья свадебных платьев (OTO)
- OTO: 3 записи мастер-класса со скидкой 60%
- Результат: OTO полностью окупил маркетинг (110к доход vs 100к затраты)
