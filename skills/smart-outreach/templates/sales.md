# Sales Templates — продажи и демо

## cold-intro — первый контакт

**Когда:** Новый потенциальный клиент. Нет предыдущего общения.

**Принципы:**
- Персонализация ОБЯЗАТЕЛЬНА — упомянуть что-то конкретное о получателе
- Ценность до продажи — сначала дать, потом предлагать
- Мягкий CTA — вопрос, не "купи"
- Представиться полным именем (Дмитрия Ростовцева)
- Не длиннее 5-6 предложений

**Психология (из marketing-psychology):**
- Reciprocity — дать ценность первым
- Curiosity gap — вызвать интерес, не раскрывая всё
- Social proof — если есть релевантный кейс

### Шаблон

```
Это агент Дмитрия Ростовцева 🤖

{personalized_hook — почему пишу именно этому человеку, 1 предложение}

{value_proposition — что мы делаем, 1-2 предложения, через призму пользы для получателя}

{soft_cta — вопрос или предложение: "Интересно было бы посмотреть, как это работает?" / "Могу показать на примере твоего бизнеса"}
```

### Пример

```
Это агент Дмитрия Ростовцева 🤖

Увидел, что ты ведёшь школу дизайна интерьера — у нас есть опыт с похожими проектами.

Мы делаем AI-агентов для онлайн-школ: они отвечают ученикам 24/7, генерируют контент, ведут базу знаний. Один из наших клиентов сэкономил ~20 часов в неделю.

Интересно посмотреть, как это может работать для твоей школы?
```

### CLEAR-ориентир: C=5 L=5 E=5 A=5 R=4 → 24/25

## warm-follow-up — повторное касание

**Когда:** Первый контакт был 3-5 дней назад, ответа нет или был нейтральный.

**Принципы:**
- Не повторять первое сообщение
- Добавить НОВУЮ ценность (кейс, инсайт, результат)
- Ещё короче, чем cold-intro
- Уважительный тон — не давить

### Шаблон

```
Это агент Дмитрия 🤖

{name}, к слову — {new_value: кейс / инсайт / результат, 1-2 предложения}

{gentle_cta — "Если интересно — покажу подробнее" / "Могу сделать мини-демо за 5 минут"}
```

### Пример

```
Это агент Дмитрия 🤖

Катя, к слову — на прошлой неделе настроили AI-агента для школы йоги. За 3 дня он обработал 80+ вопросов от учеников, которые раньше падали на администратора.

Если интересно — могу показать, как это выглядит на практике.
```

### CLEAR-ориентир: C=4 L=5 E=5 A=5 R=5 → 24/25

**Правило:** максимум 2 follow-up без ответа. После 2-го — СТОП. Вернуться только если есть новый контекст (клиент лайкнул пост, пришёл на вебинар и т.д.).

## demo-invite — приглашение на тест-драйв

**Когда:** Контакт проявил интерес (ответил, спросил, пришёл на вебинар).

**Принципы:**
- Конкретика: что получит (5 дней, полный доступ)
- Персонализированная польза — не "AI-агент", а "агент для [его ниша]"
- Без обязательств — снять риск
- Один чёткий CTA

**Психология:**
- Loss aversion — "5 дней бесплатно" = ничего не теряешь
- Endowment effect — попробовав, сложнее отказаться
- Specificity — конкретные цифры (5 дней, 50 скиллов)

### Шаблон

```
Это агент Дмитрия 🤖

{name}, подготовил для тебя тестовый доступ к Neura — {personalized_benefit}.

Тест-драйв: 5 дней, полный доступ, без обязательств.

Подключить? 🚀
```

### Пример

```
Это агент Дмитрия 🤖

Никита, подготовил для тебя тестовый доступ к Neura — AI-агент, который может вести твой контент-план, отвечать клиентам и работать с документами.

Тест-драйв: 5 дней, полный доступ к 50 навыкам, без обязательств.

Подключить? 🚀
```

### CLEAR-ориентир: C=5 L=5 E=5 A=5 R=5 → 25/25

## closing — закрытие сделки

**Когда:** Тест-драйв завершён (или завершается), клиент доволен.

**Принципы:**
- Резюме ценности — что клиент получил за время теста
- Конкретный тариф — не "от 20 000", а "тариф Старт за 20 000"
- Лёгкость перехода — "могу подключить сегодня"
- Без давления — "если вопросы — я на связи"

**Психология:**
- Status quo bias — "ты уже пользуешься, просто продолжай"
- Anchoring — сначала ценность, потом цена
- Commitment — напомнить о его интересе

### Шаблон

```
{name}, по итогам тест-драйва:

{summary_of_value — что бот сделал за 5 дней, 2-3 пункта}

{tariff — "Тариф Старт: 20 000 ₽/мес — всё, чем ты пользовался + поддержка"}

Если вопросы — я на связи. Если готов — могу подключить сегодня.
```

### Пример

```
Юлия, по итогам тест-драйва:

За 5 дней твой агент:
— Обработал 45 запросов
— Создал 3 документа по твоим инструкциям
— Работал 24/7 без перерывов

Тариф Старт: 20 000 ₽/мес — все 50 навыков + поддержка и обновления.

Если вопросы — я на связи. Если готова — подключу сегодня.
```

### CLEAR-ориентир: C=5 L=5 E=5 A=5 R=5 → 25/25

**Примечание:** closing пишется без "Это агент Дмитрия 🤖" — к этому моменту контакт уже знает кто пишет. Интро создаёт дистанцию, которая мешает закрытию.
