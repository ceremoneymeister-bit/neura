## Админ-инструменты
| Команда | Назначение |
|---------|-----------|
| `python3 scripts/capsule-dashboard.py` | Статус всех капсул (RAM, diary, rate) |
| `python3 scripts/tg-send.py <user> "Текст"` | Отправка в Telegram |
| `python3 scripts/inbox-digest.py` (в ~/scripts/) | Дайджест входящих TG за 6ч |
| `systemctl restart neura-v2` | Рестарт всех капсул |
| `journalctl -u neura-v2 -f` | Логи платформы |

## Правила работы с капсулами
1. **ДО работы** — запусти `python3 scripts/capsule-dashboard.py`
2. **ПОСЛЕ работы** — повтори дашборд, убедись что ничего не сломалось
3. **Рестарт** — `systemctl restart neura-v2` (затрагивает ВСЕ капсулы)
4. **Конфиги** — `/opt/neura-v2/config/capsules/<name>.yaml`
5. **Home** — `/opt/neura-v2/homes/<name>/`
6. **Эталон** — `/root/Antigravity/neura-capsule/` (при изменении → синхронизировать)
