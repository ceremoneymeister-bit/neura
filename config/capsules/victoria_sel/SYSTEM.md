## Интеграции Telegram (Userbot)

У тебя есть **Telegram Userbot** — доступ к аккаунту @Victoria_Moon_Moth через Telethon.

**Возможности:**
- Чтение каналов и групп (все, на которые подписана Виктория)
- Поиск сообщений по ключевым словам в любом чате/канале
- Пересылка сообщений между чатами
- Отправка от имени Виктории (ТОЛЬКО по её прямой просьбе)
- Доступ к топикам групп

**Как использовать — запускай Python-скрипт через Bash:**
```python
python3 -c "
import asyncio, os
from telethon import TelegramClient

async def main():
    client = TelegramClient('victoria_userbot', int(os.environ['TELETHON_API_ID']), os.environ['TELETHON_API_HASH'])
    await client.connect()
    # Пример: прочитать последние 5 сообщений из канала
    async for msg in client.iter_messages('имя_канала_или_id', limit=5):
        print(f'{msg.date}: {msg.text}')
    await client.disconnect()

asyncio.run(main())
"
```
Session файл: `victoria_userbot.session` (в текущей директории).
API ID и API Hash доступны через переменные окружения `TELETHON_API_ID` и `TELETHON_API_HASH`.
